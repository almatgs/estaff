Integratsioon Microsoft Outlook kalendriga

Kavandatud sündmuste impordiks (reeglina, Töövestlus ning Töövestlus tellijaga) Microsoft Outlook kalendrisse ning (või) nende ekspordiks sellest tuleb teostada alljärgnevat:

  • Vajutage Servis->Personaalsed seadistused
  • Avaneb punkt Kalender. Tähistage MS Outlook välise kalendrina.
  • Avage punkt Väliste tarkvarade moodulid. Kirje Microsoft Outlook vastas vajutage nupp Paigaldada.

Peale seda imporditakse planeeritud töövestlus automaatselt MS Outlook kalendrisse. Selle sündmuse kuupäeva/kellaaja muutmisel Datex Recruit kalendris muutub see samuti ka MS Outlook-s.

  • Kui on vajalik, et sündmus Töövestlus tellijaga oleks imporditud MS Outlook-i koos lisatud kandidaadi CV-ga, siis on võimalik seadistada eksporti malli järgi. Sellisel juhul laaditakse Outlook-i sündmuse kaart koos lisatud CV failiga. Selleks on vajalik luua spetsiaalset kirja malli ekspordi jaoks (täpsemalt tööst mallidega – sektsioonis Kirjade mallid). Loodavas mallis määrake Saaja tüüp – Kirje välises kalendris ning täitke väljad Kirja teema ning Kirja tekst. Kirja tekstis sisaldavat informatsiooni kajastatakse sündmuse kaardis MS Outlook selle nimetusena. Soovil võib kasutada nii teemas, kui ka kirja tekstis erinevaid arvutatavaid elemente. Kirje Kaardile lisatud failid vastas vajutage nupp „+“ (Lisada), määrake faili nimetus (soovitatakse sisaldada arvutatavat elementi Kandidaat: Perekonnanimi, Nimi), manuse tüüp - CV, esialgne objekt - Kandidaat. Seejärel sektsioonis Administreerimine–> Kataloogid–> Üldised–> Sündmuste tüübid avage sündmuse kaart Töövestlus tellijaga ning vahelehes Lisasündmused valige sündmuse kaardi loomise mallina välises kalendris eelnevalt loodud mall.

 

  • Kavandatud sündmuste ekspordiks MS Outlook kalendrist Datex Recruit sisemisse kalendrisse kasutatakse taustaülesannet Import välisest kalendrist. Selle taustaülesande seadistamine toimub Administreerimine-> Kataloogid–> Muud-> Lokaalsed taustaülesanded.

Manuaalses režiimis käivitatakse ülesannet tarkvara peamenüüst käsuga Servis-> Taustaülesande manuaalne käivitus -> Import välisest kalendrist.

Regulaarseks impordi seadistamiseks (igapäevaselt või muu ajalise perioodi pärast) märkige vajalikud väärtused väljadel Tund  ja Minut.

Sündmuste impordil Microsoft Outlook kalendrist imporditakse erasündmustena märgistatud objektid ilma teema- ja kirjelduseta.

Planeeritud kuupäeva/kellaaja muutmisel MS Outlook kalendris muutub see samuti ka Datex Recruit kalendris (peale taustaülesande Import välisest kalendrist töötlemist manuaalses või automaatses režiimis).
