Impordi stsenaariumid

Impordi stsenaarium võimaldab importida tarkvara:

  • Eelnevalt Datex Recruit süsteemist eksporditud ning esitletud, reeglina, sisemises XML-formaadis andmeid erinevate tüüpide objektide kohta (struktuurüksused, kandidaadid, vakantsid jne.),
  • Eelnevalt suvalise tüübi välisest süsteemist eksporditud ning esitletud XML, CVS või Microsoft Excel (XLS) formaadis andmeid erinevate tüüpide objektide kohta

 

Eelnevalt Datex Recruit süsteemist eksporditud andmete import:

  • Impordi uue stsenaariumi seadistuste kaardi loomiseks avage sektsioon Administreerimine–>Kataloogid–>Integratsioon–>Impordi stsenaariumid ning vajutage nupp Uus element.
  • Sisestage stsenaariumi nimetus, paigaldage lüliti Andmete import sisemises formaadis peale.
  • Nupu Ülevaade abil määrake ligipääs andmetega lähtekaustale impordiks.
  • Juhul, kui on vajalik importida mitte kõiki objekte, vaid ainult uusi (et vältida sisemise andmebaasi moonutamist ühtivate objektide vananenud andmetega), paigutage lipp Importida ainult uusi andmeid.
  • Juhul, kui on vajalik importida mitte kõiki objekte, vaid ainult vajaliku tüübi objekte, vajutage nupp Lisada imporditavat objekti ning valige vajalik objekt nimekirjast.

Andmete import Datex Recruit süsteemi välistest süsteemidest:

  • Impordi uue stsenaariumi seadistuste kaardi loomiseks avage sektsioon Administreerimine–>Kataloogid–>Integratsioon–>Impordi stsenaariumid ning vajutage nupp Uus element.
  • Sisestage stsenaariumi nimetus, paigaldage lüliti Andmete import teistest tarkvaradest peale.
  • Valige lähtefaili formaat. Formaadi CSV või Microsoft Excel valimise juhul, kontrollige üle lipu Faili esimene rida sisaldab väljade nimetusi seisukorda teie failide esimesele reale vastavuse osas.
  • Juhul, kui informatsioon on esitatud ühes failis, paigutage lüliti Import ühest failist peale, kui aga mitmes – siis Import failide kaustast peale.
  • Nupu Ülevaade abil määrake ligipääs andmetega lähtekaustale impordiks.
  • Juhul, kui impordifailide esimene rida sisaldab väljade nimekirju, siis neid saab juurde laadida nupuga Importida väljade nimekirja. Väljade nimetused võivad olla suvalised, kui kirjutatud ladinatähtedega.
  • Iga imporditud välja jaoks sisestage imporditava objekti vastava sisemise välja nimetus.
  • Imporditavate objektide vajaliku tüübi määramiseks vajutage nupp Lisada imporditav objekt ning valige vajalik nimekirjast.

 

  • Manuaalses režiimis käivitatakse impordi stsenaariumi tarkvara peamenüüst käsuga Servis–> Import-> [stsenaariumi impordi nimetus].
  • Automaatse impordi seadistamiseks vajutage nupu Taustaülesanded vastas nupp „+“ (Luua taustaülesannet), sisestage ülesande nimetus ning määrake käivitamise ajakava režiim (Igapäevaselt määratud ajal / Võrdsete ajavahemike pärast, ajalised parameetrid sisestage vastavalt väljades Tund, Minut/ ajavahemik).
