Datex Recruit laseb lisada nuppu Import Datex Recruit-sse  Internet Explorer, Mozilla Firefox või Google Chrome instrumentide paneelile. Selle nupu abil võib kiiresti tõsta kandidaadi CV-d sisaldava veebilehekülje fragmenti Datex Recruit rakendusse.

Esimesel käivitamisel pakub Datex Recruit lisada sellist nuppu automaatselt. Samuti on võimalik teha seda hiljem sektsioonis Personaalsed seadistused ->Moodulid väliste tarkvarade jaoks.

 

Internetimooduli poolt toetavatel veebilehtedel kandidaadi CV kiireks tõstmiseks veebilehelt Datex Recruit rakendusse on piisav nupu Import Datex Recruit-sse vajutamisest Teie brauseri instrumentide paneelil. Teistel veebilehtedel enne selle nupu vajutamist on vajalik selekteerida kandidaadi CV-d sisaldava lehe osa.

Avaneb kandidaadi kaart, kus on juba täidetud tarkvara poolt tuvastatud väljad. Ekraani alumises osas asub kandidaadi CV esialgsel kujul.

Enne CV lisamist teostab rakendus eelnevalt kontrolli võimalike dublikaatide olemasolu osas andmebaasis. Juhul, kui avastatakse dublikaate, annab tarkvara ette vastavat teavitust
