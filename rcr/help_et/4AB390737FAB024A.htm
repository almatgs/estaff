NTLM-autentimise seadistus

Juhul, kui Datex Recruit  kasutajate tööjaamade arvutid  on ühendatud võrguga, mille koosseisu kuulub domeeni kontroller Active Directory, siis Datex Recruit süsteemi saab seadistada NTLM-autentimise töörežiimiks.

NTLM-autentimise kasutamiseks süsteemis Datex Recruit Recruiter tuleb teostada alljärgnevat:

Arvutil, kus on paigaldatud Datex Recruit server:

· Datex Recruit serveri kataloogi juures luua faili xHttp1.ini ning lisada sellesse kirje

LDS-NTLM-AUTH: 1

·Käivitage "Datex Recruit Server" menüüst Start->Tarkvarad (Start->Programs)

·Väljas Kasutajanimi sisestage seadistusi teostava tehnilise süsteemadministraatori nimi formaadis domain\user, kus domain ja user – on vastavalt Active Directory domeeni nimetus ning kasutajanimi, välja Salasõna tuleb tühjendada.

·Käivituge server Datex Recruit uuesti, vajutades nuppu Lülitada server ning seejärel – Lülitada server sisse.

Klientide suvalisel töökohal

· Datex Recruit kliendikoha kataloogi juures looge fail SpXml2.ini ning lisage sellesse kirje

EXTERNAL-LDS: 1

·Käivitage "Datex Recruit Server" menüüst Start->Tarkvarad (Start->Programs)

·Sisestage tehnilise administraatori nimi (formaadis user) ning salasõna. Salasõnana kasutage vastava Active Directory konto salasõna.

·Sektsioonis Administreerimine->Ligipääs->Kasutajad määrake iga kasutaja jaoks jooksva nime asemel Active Directory  kasutajanimi (ainult login, kusjuures kui selles on olemas suurtähed – vahetage neid välja väiketähtede vastu), välja Salasõna tuleb tühjendada.

·Sulgege tarkvara, seejärel käivitage see uuesti. Tarkvara peab avanema automaatselt tegeliku arvutil töötava kasutaja all. Kui see ei toimi, tuleb ära muuta Internet Explorer brauseri seadistusi. Brauseri omadustes vahelehel Turvalisus tsooni jaoks, mis vastab kliendikoha asukohale, vajutage nupp Muu (Custom level). Turvalisuse parameetrite seadistuste aknas sektsioonis Kasutaja autentimine->Sissepääs (User Authentication->Logon) paigaldage lüliti seisukorda Automaatne sissepääs jooksva kasutajanimega ning salasõnaga (Automatic logon with current user name and password).

Ülejäänud klientide töökohtadel:

·Datex Recruit kliendikoha kataloogi juures looge fail SpXml2.ini ning lisage sellesse kirje

EXTERNAL-LDS: 1

·Käivitage „Datex Recruit" menüüst Start->Tarkvarad (Start->Programs). Tarkvara peab avanema automaatselt tegeliku arvutil töötava kasutaja all. Kui see ei toimi, tuleb ära muuta Internet Explorer brauseri seadistusi, täpselt samamoodi, nagu see on ülespool kirjeldatud.
