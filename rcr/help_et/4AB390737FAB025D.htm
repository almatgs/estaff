Erialad

Vaikimisi kasutab tarkvara tarkvaraga koos tarnitavat kahetasemelist erialade kataloogi.

Teil on võimalik käsitsi redigeerida standardset erialade kataloogi sektsioonis Administreerimine–>Kataloogid–>Üldised–>Erialad.

Juhul, kui erialade kataloog on esitletud Microsoft Excel (XLS) formaadis, on seda võimalik moodustada automaatselt, importides andmeid sellisest xls-failist  Datex Recruit-i.

Hoiatus! Kirjeldatud impordi teostamisel erialade kataloogi standardseid elemente kustutakse.

  • Erialade kataloogi impordiks xls-failist on vaja siseneda sektsiooni Administreerimine–>Kataloogid–>Üldised–>Erialad ning vajutada nuppu Import MS Excel-st.
  • Avanenud aknas tuleb määrata xls-faili ning vajutada nuppu Avada.
  • Käivitub import ning selle lõppemisel ilmub sektsioonis Erialad imporditud erialade kataloog.

XLS faili formaat erialade impordiks

Faili esimene veerg on juurveeruks, s.t. see võib sisaldada eriala ja (või) kategooria nimetust (hierarhilise erialade kataloogi ülesehitamiseks). Iga järgnevatest veergudest on alluvaks kataloogiks, s.t. see võib sisaldada reastatud erialade komplekti (hierarhilise puu lõpetamine) ja (või) alamkategooria nimetust (hierarhia süvendamine).
