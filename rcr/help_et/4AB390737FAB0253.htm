Datex Recruit serveriosa tõstmine teisele arvutile

  • Käivitage "Datex Recruit Server" menüüst Start -> Tarkvarad (Start -> Programs) ning lülitage Datex Recruit server välja.
  • Desaktiveerige server, kasutades menüüd Servis-> Desaktiveerida. Sulgege Datex Recruit serveri aken.
  • Arhiveerige kaust, kuhu on paigaldatud Datex Recruit server (vaikimisi - C:\Program Files\DatexRecruit_Server), kopeerige arhiiv uuele arvutile ning tõstke kaust DatexRecruit_Server arhiivist lahti analoogsesse kohta.
  • Paigaldage server uuele arvutile paigalduskomplektist DatexRecruit_Server.exe. Paigaldusel veenduge, et serverit paigaldatakse samasse kausta (DatexRecruit_Server), mille olete kopeerinud vanast arvutist.
  • Käivitage „Datex Recruit Server" menüüst Start -> Tarkvarad (Start -> Programs), lülitage sisse ning aktiveerige Datex Recruit server.
