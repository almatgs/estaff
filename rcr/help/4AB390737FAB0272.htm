Автоматическая отправка почты

Раздел Автоматическая отправка почты предназначен для настройки функции автоматической отправки почты, используемой как для отправки единичных писем (при отсутствии на рабочих местах почтовой программы), так и для массовой отправки писем по различным шаблонам кандидатам или контактным лицам. Кроме того, данный функционал может использоваться для отправки из программы различных уведомлений.

Для правильной настройки функции автоматической отправки почты в E-Staff может потребоваться помощь Вашего системного администратора. В отдельных случаях настройку сможет произвести только системный администратор.

  • В поле Сервер SMTP введите сетевое имя или IP-адрес почтового сервера.
  • Если на почтовом сервере используется авторизация пользователей, то установите флажок Сервер требует авторизации и введите логин и пароль.
  • Для отправки писем c использованием протокола шифрованных (защищенных) соединений  установите флажок Использовать SSL (требуется TLS) либо Выделенный SSL-порт (SSMTP, требуется SSL).
  • В полях Адрес отправителя и Имя отправителя введите электронный адрес (с которого будут отправляться письма) и имя соответственно. Если необходимо, чтобы письма при массовой рассылке отправлялись с каждого клиентского места от имени работающего на нем рекрутера, то в разделе Персональные настройки–>Технические–>Отправка почты в полях Адрес отправителя и Имя отправителя введите его электронный адрес (e-mail) и имя.
  • Флажок Отправлять с сервера приложения инициирует отправку писем непосредственно с сервера E-Staff, а не с клиентского места.
  • Для проверки правильности введенных данных в поле Тестовый E-mail укажите свой электронный адрес и нажмите кнопку Отправить тестовое сообщение. При корректно введенных данных на указанный почтовый адрес придет сообщение с текстом Test.
  • Сохраните изменения.

Для использования функции автоматической отправки почты для отправки единичных писем разделе Сервис->Персональные настройки->Технические->Отправка почты необходимо указать метод отправки SMTP.
