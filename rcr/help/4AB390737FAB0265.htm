Карточка квартального бюджета

Карточка квартального бюджета в программе E-Staff Рекрутер имеет следующую структуру:

Поле Год содержит год, на который производится поквартальное закладывание бюджета.

Поле Квартал содержит ссылку на номер квартала (значение из справочника).

Поле Плановая сумма содержит общую сумму средств, выделенных на указанный квартал согласно бюджету.

Поле Комментарий может содержать произвольную дополнительную информацию по квартальному бюджету.
