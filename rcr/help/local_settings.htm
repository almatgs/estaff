Персональные настройки

Пользовательские настройки задаются для каждого пользователя (компьютера) и могут произвольно меняться каждым пользователем без ущерба для работы остальных пользователей и системы целиком.

Для открытия окна редактирования пользовательских настроек выберите команду Персональные настройки из меню Сервис.

Окно пользовательских настроек состоит из разделов, описанных ниже.
