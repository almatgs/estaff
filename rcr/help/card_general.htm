Общие принципы работы с карточками

Карточки всех типов объектов программы (кандидатов, вакансий, и т.д.) построены по единому принципу. Открываемые карточки появляются на экране в виде отдельных окон. Размер окна карточки можно изменять. Программа запоминает размер окон для каждого типа объектов, поэтому при последующем открытии карточки будут иметь выбранный ранее размер.

Наличием и расположением (для отдельных полей возможно указать, в каком разделе показывать поле – “Общие сведения” или “Доп. сведения”) стандартных полей в карточках, а также возможностью установки выбранных полей карточек в качестве обязательных к заполнению можно управлять с помощью раздела Администрирование->Общие настройки->Управление полями карточек.

В программе предусмотрена возможность массовой смены значения поля в карточках. Выполняется следующим образом:

  • Выделите в списке все карточки, в которых требуется сменить значение какого-либо поля на новое
  • В главном меню вызовите команду Действия->Установить значение поля
  • В выпадающем списке поля «Поле» выберите требуемое поле, а в поле «Значение» укажите новое значение и нажмите ОК.

Карточка каждого кандидата, как правило, содержит один прикрепленный документ - резюме. Помимо этого, Вы можете прикреплять произвольное количество документов к карточке кандидата, а также вакансии или заказчика (например, резюме на иностранном языке, подробное описание вакансии, договор с заказчиком и т.д.).

Для этого в открытой карточке на панели инструментов нажмите кнопку Добавить приложение ("Скрепка"), выберите тип документа и укажите один файл либо несколько файлов (несколько файлов можно добавлять при включенной опции Разрешить прикрепление нескольких файлов в настройках данного типа документа в разделе Администрирование->Справочники->Общие->Приложения к карточкам). Впоследствии при необходимости в такое приложение можно еще добавлять файлы с помощью кнопки Загрузить, а также убирать их командой Удалить в контекстном меню.

Если соответствующий тип документа в списке отсутствует, выберите пункт Прикрепить файл.

Для прикрепленных файлов Microsoft Word, Microsoft Excel или Adobe Reader предварительный просмотр их содержимого возможен непосредственно в карточке без открытия файла.

После выбора типа документа откроется страница с пустым документом. Вы можете вводить текст документа вручную, скопировать его из другого редактора, или импортировать напрямую из файла Microsoft Word, OpenOffice или Adobe Reader, нажав кнопку Импорт.

Для удаления прикрепленного документа/файла из раздела Приложения, необходимо выбрать сам раздел, затем в правой части окна выделить удаляемый документ/файл и на клавиатуре нажать кнопку «Delete».

При закрытии новой заполненной карточки любого объекта, или карточки, в которую были внесены изменения, программа предложит сохранить новые данные в базе. Выберите ответ Да. Также Вы можете сохранить данные заранее, выбрав команду Сохранить из меню Файл (или нажав Ctrl+S на клавиатуре).
