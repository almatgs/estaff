Приложения к карточкам

Справочник Приложения к карточкам содержит список используемых в системе приложений к карточкам объектов. Вместе с программой поставляются несколько стандартных приложений, наиболее часто используемых при работе с теми или иными объектами, но при необходимости Вы можете редактировать существующие и добавлять собственные.

  • Для добавления нового приложения нажмите кнопку Новый элемент
  • В открывшемся окне введите Наименование приложения.
  • Выберите Базовый объект для указания принадлежности к тому или иному объекту, определяющей, в каких карточках возможно будет добавлять такое приложение.

Опция Добавлять автоматически при создании новой карточки означает автоматическое формирование приложения данного типа (при необходимости - по заданному начальному шаблону) при создании карточки указанного базового объекта (вакансии,  кандидата, организации или физического лица).

  • Укажите необходимый Тип приложения (обычный Текст, Форматированный текст или Файл).

В случае выбора типа приложения Форматированный текст появится возможность связки создаваемого приложения с дополнительными шаблонами.

Опция Разрешить прикрепление нескольких файлов включает возможность добавления приложения данного типа, включающего в себя не один, а несколько файлов. Опция доступна только для приложений типа “Файл”.

Опция Шаблон MS Word для печати и отправки по e-mail позволяет прикрепить, например, фирменный бланк организации, чтобы при распечатывании документа или отправке письма по электронной почте это приложение формировалось на фирменном бланке.

Опция Начальный шаблон позволяет задать некоторый начальный шаблон, чтобы при добавлении приложения в карточке объекта оно формировалось на основе этого шаблона.
