Отчеты

Модуль “Массовый подбор” включает в себя следующие отчеты:

  • Отчет “Штатное наполнение” позволят видеть картину заполнения штатных должностей по подразделению массового подбора.
  • Отчет “Воронка подбора” показывает региональную статистику по процессу массового подбора, позволяющую оценить отток на каждом из этапов, от первичного контакта до испытательного срока.

По каждому показателю в отчетах можно посмотреть детализацию статистики по какому-либо показателю. Для этого необходимо щелкнуть по заполненной ячейке отчета.
