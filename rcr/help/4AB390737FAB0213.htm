Добавление кандидата вручную

Добавление кандидата вручную является достаточно редкой операцией. Тем не менее, иногда  в ней может возникнуть потребность.

  • Для добавления кандидата вручную нажмите кнопку Новый кандидат на панели инструментов.

    Откроется пустая карточка кандидата.

  • Заполните те поля на странице Общие сведения, по которым у Вас есть информация, и закройте карточку.

    Подробнее о заполнении полей карточки можно прочитать в разделе Карточка кандидата.

    При закрытии карточки кандидата программа предложит сохранить данные о кандидате в базе. Выберите ответ Да.

    Также Вы можете сохранить данные о кандидате заранее, выбрав команду Сохранить из меню Файл (или нажав Ctrl+S).
