SMS-шлюзы

Для правильной настройки данного раздела может потребоваться помощь Вашего системного администратора. В отдельных случаях настройку сможет произвести только системный администратор.

Добавление SMS-шлюзов производится в разделе Администрирование–>Справочники–>Интеграция–>SMS-шлюзы с помощью кнопки Новый элемент в правом верхнем углу.

Поле Наименование используется только для визуального различия между несколькими агрегаторами и может содержать произвольный текст.

Поле Скрипт для отправки SMS предназначено для ввода программного кода сценария, который разрабатывается индивидуально под определенного SMS-агрегатора, с которым Вы работаете.

Для проверки правильности введенных данных в поле Тестовый номер телефона укажите номер мобильного телефона и нажмите кнопку Отправить тестовое сообщение. При корректно введенных данных на указанный номер придет SMS-сообщение с текстом Test.
