Карточка затраты

Карточка затраты в программе E-Staff Рекрутер имеет следующую структуру:

Поле Контрагент содержит ссылку на контрагента, с которым заключен договор.

Поле Статья затрат содержит ссылку на статью затрат (выбирается из настраиваемого справочника статей затрат).

Справочник статей затрат заполняется пользователем в разделе Администрирование->Справочники->Общие->Статьи затрат.

Поле Договор содержит ссылку на номер договора, заключенного с контрагентом.

Поля Период оказания услуги (с, по) содержит даты начала и окончания оказания предоставляемых услуг, оговоренных в договоре.

Поля Структурное подразделение, Вакансия, Кандидат содержат соответствующие ссылки для указания объекта, с которым связана данная затрата.

Поле Сумма платежа содержит общую сумму платежа по данной затрате.

Поле Валюта содержит ссылку на вид денежной валюты, используемой для оплаты (выбирается из справочника валют).

Поле Статус платежа содержит ссылку на статус совершаемого платежа (выбирается из настраиваемого справочника статусов платежей).

Поле Дата оплаты содержит дату совершенного платежа по данной затрате.

Поле Номер платежа содержит номер платежного документа.

Поле Доп.сведения может содержать произвольную дополнительную информацию по данной затрате.
