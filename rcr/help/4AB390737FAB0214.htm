Вставка резюме из буфера

Вставка резюме из буфера является одним из способов одиночного добавления кандидата. Данный способ обычно используется, когда другими способами затруднительно воспользоваться (например, резюме хранится в программе, для которой не предусмотрено добавление специальной кнопки для импорта резюме, или файл резюме имеет формат, неподдерживаемый программой E-Staff). Т.е. у Вас перед глазами открыто резюме кандидата, которое необходимо занести в базу.

  • Для переноса открытого в данный момент резюме в программу выделите и скопируйте текст резюме (используя команду Копировать из меню Правка или контекстного меню, вызываемого правой кнопкой мыши, либо сочетание клавиш Ctrl+C)
  • Перейдите в E-Staff и нажмите кнопку Вставка резюме на панели инструментов.

При этом программа автоматически извлечет ключевые поля из резюме и откроется карточка кандидата, в которой будут заполнены поля, которые программа смогла распознать. В нижней части карточки в окне предварительного просмотра будет расположено прикрепленное к карточке кандидата резюме в исходном виде.

Подробнее о заполнении полей карточки можно прочитать в разделе Карточка кандидата.

При закрытии карточки кандидата программа предложит сохранить данные о кандидате в базе. Выберите ответ Да.

Также Вы можете сохранить данные о кандидате заранее, выбрав команду Сохранить из меню Файл (или нажав сочетание клавиш Ctrl+S).
