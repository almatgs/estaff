Статистика

Раздел Статистика открывается в отдельном окне командой Сервис->Статистика или нажатием на панели инструментов кнопки Статистика (в виде диаграммы). 

Данный раздел содержит ряд отчетов, сгруппированных по типу или объекту отчетности. Для формирования того или иного отчета необходимо выбрать его из списка в левой части окна, указать при необходимости дополнительные параметры (Дата, Рекрутер и т.д.) и нажать кнопку Показать. Для фильтров, ссылающихся на справочник (например, Направление подбора, Текущий статус), можно выбрать сразу нескольких значений, поочередно указывая их, удерживая клавишу Ctrl. Для быстрого выбора стандартного периода (неделя, месяц, квартал, год) можно использовать дополнительную кнопку, расположенную справа от поля Дата.

Некоторые отчеты позволяют посмотреть детализацию статистики - подробные данные в виде выборки из базы. Для получения детализации статистики по какому-либо показателю необходимо щелкнуть по заполненной ячейке отчета.

Любой из отчетов можно экспортировать в Word кнопкой с изображением принтера ("Печать"), либо в Excel кнопкой "Экспорт в Microsoft Excel". Кроме того, в Excel отчет может быть экспортирован в заранее подготовленный шаблон путем автоматической вставки ячеек отчета на первый лист. Подготовленный шаблон прикрепляется в настройках отчета в разделе Администрирование->Сервис->Отчеты.

Текущее состояние

Текущие вакансии по рекрутерам - отчет, отражающий по каждому рекрутеру список вакансий, относящихся к выбранному направлению подбора. По каждой вакансии будет показано структурное подразделение (с иерархической разбивкой), количество позиций (общее и открытых), дата открытия, сколько дней она находится в работе.

Текущие вакансии по подразделениям - отчет, отражающий по каждому структурному подразделению список вакансий, относящихся к выбранному направлению подбора. По каждому структурному подразделению будет показан список вакансий, поступивших от него, и для каждой вакансии будет показано  количество позиций (общее и открытых), дата открытия, сколько дней она находится в работе.

Текущие вакансии по клиентам - отчет, отражающий по каждому клиенту список вакансий, относящихся к выбранному направлению подбора. По каждому клиенту будет показан список вакансий, поступивших от него, и для каждой вакансии будет показано  количество позиций (общее и открытых), дата открытия, сколько дней она находится в работе.

Текущая загрузка рекрутеров - отчет, отражающие текущую загруженность рекрутеров при работе по вакансиям, относящихся к выбранному направлению подбора. По каждому из рекрутеров будет показано общее количество вакансий, находящихся у него в работе, текущая загрузка и количество просроченных вакансий.

Текущий прогресс - отчет, отражающий ход процесса работы над вакансиями выбранного направления подбора. По каждой вакансии  будет показано количество дней в работе, количество публикаций на работных сайтах, сколько кандидатов было "на входе" (количество поступивших откликов и кандидатов, найденных самостоятельно), сколько из них было отобрано и сколько дошло до каждого из этапов, начиная от планирования интервью и заканчивая оффером.

 

Процесс подбора

Воронка подбора - отчет, отражающий ход процесса работы над вакансиями за указанный период. По каждой вакансии будет сформирована "воронка подбора", отражающая приток соискателей на входе каждого из этапов подбора и соответствующий отток на выходе (как в количественном, так и в процентном выражении).

Отказы - отчет, позволяющий оценить объемы оттока кандидатов, не прошедших отбор, за требуемый период по вакансиям заданного направления подбора. По каждой вакансии будет показано, сколько всего было кандидатов в работе и сколько из них не прошли отбор по причине того или иного отказа (как в количественном, так и в процентном выражении).

Причины самоотказа кандидатов - отчет, позволяющий проанализировать причины самоотказов соискателей за требуемый период по вакансиям заданного направления подбора и/или категории персонала. По каждой вакансии будет показано, сколько всего было самоотказов и сколько из них отказались по того или иной причине (как в количественном, так и в процентном выражении).

Отток на испытательном сроке - отчет, позволяющий оценить объемы оттока кандидатов, принятых на работу за выбранный период, но не прошедших испытательный срок, и рассматриваемых на вакансиях заданного направления подбора и/или категории персонала. По каждому рекрутеру будет показано, сколько всего кандидатов им было принято на работу и сколько из них было уволено (как в количественном, так и в процентном выражении).

Процесс подбора 1 - отчет, отражающий ход процесса работы над вакансиями за указанный период (Период). По каждой вакансии, по которой велась какая-либо работа с кандидатами в указанном периоде, будет показана, так называемая "воронка подбора": сколько кандидатов было "на входе" (отобрано), сколько из них прошли интервью, сколько резюме (прошедших интервью кандидатов) было отправлено на рассмотрение заказчику (дополнительно в столбце «%» - их доля по отношению к количеству состоявшихся интервью), сколько из них в дальнейшем прошли интервью с заказчиком (дополнительно в столбце «%» - их доля по отношению к количеству резюме, рассмотренных заказчиком), сколько из них в результате было взяты на работу. Последние три столбца показывают, сколько из отобранных кандидатов было отклонено тем или иным инициатором: рекрутером, заказчиком или соискателем (самоотказ).

Процесс подбора 2 - аналогичный отчет, отражающий ход процесса работы над вакансиями за указанный период (Период), но дополнительно содержащий информацию о количестве активных вакансий, их приоритетах, перечень финальных кандидатов и комментарии к вакансии. Заключительным этапом "воронки подбора" данного отчета является интервью с заказчиком (для рекрутинговых агентств) либо проверка службой безопасности (для кадровых служб).

Вакансии + Кандидаты - отчет, отражающий текущее положение дел по активным вакансиям рекрутера/группы в части работы с кандидатами. По каждой активной вакансии, кроме основной информации (дата открытия, количество активных вакансий, структурное подразделение (для кадровых служб) либо клиент (для рекрутинговых агентств)) будет показан актуальный список кандидатов (т.е. только тех, которые находятся в активном статусе и с которыми за последние два месяца велась какая-либо работа). Для каждого кандидата будет указан его текущий статус.

Рекрутеры + события – сводный отчет, отражающий общую картину и эффективность по работе всех рекрутеров за указанный период на основании их активности. По каждому рекрутеру, которым велась какая-либо работа с кандидатами в указанном периоде, будет показана "воронка подбора", начиная с количества кандидатов, занесенных в базу, до количества принятых на работу, отклоненных или включенных в резерв.

 

Вакансии

Состояние вакансий по рекрутерам - отчет, отражающий количественные показатели рекрутеров по работе с вакансиями за указанный период. По каждому из рекрутеров будет показано количество вакансий на начало/конец периода, количество открытых, закрытых, приостановленных и отозванных в течение заданного периода, а также количество вакансий, планируемых к закрытию.

Состояние вакансий по подразделениям - отчет, отражающий количественные показатели по работе с вакансиями, поступившими из структурных подразделений, за указанный период. По каждому из структурных подразделений будет показано количество вакансий на начало/конец периода, количество открытых, закрытых, приостановленных и отозванных в течение заданного периода, а также количество вакансий, планируемых к закрытию.

Состояние вакансий по клиентам - отчет, отражающий количественные показатели по работе с вакансиями, поступившими от клиентов, за указанный период. По каждому из клиентов будет показано количество вакансий на начало/конец периода, количество открытых, закрытых, приостановленных и отозванных в течение заданного периода, а также количество вакансий, планируемых к закрытию.

Общие показатели – отчет, отражающий общие показатели по вакансиям, открытым за указанный период (Дата открытия): сколько было в целом открыто вакансий, сколько из них было закрыто, в т.ч. в срок, сколько отменено, средний срок закрытия вакансий.

Сроки поиска - отчет, отражающий результаты работы по вакансиям, открытым за указанный период (Дата открытия). По каждой вакансии будет показано, сколько дней она находится в работе, нормативный срок, плюсом будут отмечены вакансии, закрытые в срок, и по каждому пользователю посчитается процент вакансий, закрытых в срок. Кроме того, будет показан финальный кандидат и в самом последнем столбце - комментарий к вакансии, например, уважительная причина, по которой вакансия не была закрыта в срок. Если требуется посмотреть отчет только по закрытым вакансиям, необходимо переключить динамический фильтр Текущий статус на Закрыта.

Итоговые показатели:

В строке группировки по рекрутеру:

- столбец "Дней в работе" - среднее количество дней в работе по вакансиям.

- столбец "+" - количество вакансий, закрытых в срок.

В строке Итого:

- столбец "Текущий статус" - суммарное количество вакансий отчета.

- столбец "Дней в работе" - среднее количество дней в работе по всем вакансиям отчета.

- столбец "+" - суммарное количество вакансий, закрытых в срок

Сроки поиска (детализация) - аналогичный отчет, отражающий результаты работы по всем вакансиям (включая детализированные), открытым за указанный период (Дата открытия).

Закрытые по датам - отчет, отражающий ключевую информацию по вакансиям, закрытым за указанный период (Дата закрытия). По каждой вакансии будет показаны дата открытия и закрытия, структурное подразделение (для кадровых служб) либо клиент (для рекрутинговых агентств) и финальный кандидат.

Итоговые показатели: подсчитывается количество закрытых вакансий за месяц, год и общее их количество.

По источникам подбора - отчет, отражающий эффективность источников поступления кандидатов (например, сайтов, кадровых агентств) "на выходе", т.е. позволяющий оценить, из какого из них поступает больше кандидатов, в результате закрывающих собой вакансию.

Итоговые показатели: подсчитывается количество закрытых вакансий.

По уровням сложности, По специальностям, По рекрутерам/группам - отчеты, отражающие текущие результаты работы по вакансиям, открытым за указанный период (Дата открытия), в разрезе уровней сложности, специальностей или рекрутеров соответственно. По каждому из параметров будет показано общее количество вакансий, открытых в течение заданного периода, сколько из них было закрыто по состоянию на текущий момент, сколько из закрытых было закрыто в срок, средний срок поиска и количество отмененных вакансий.

Итоговые показатели: подсчитывается общее количество вакансий по каждому показателю и средний срок поиска.

Кандидаты

По источникам поступления - отчет, отражающий эффективность источников поступления кандидатов "на входе", т.е. позволяющий оценить, из какого из них поступает больше кандидатов.

Итоговые показатели: подсчитывается общее количество внесенных в базу кандидатов и принятых на работу.

По специальностям - отчет, отражающий распределение кандидатов, добавленных рекрутером в базу за указанный период (Дата внесения кандидата), по специальностям.

Внутренние рекомендации - отчет, отражающий список кандидатов, принятых на работу за указанный период (Дата) из числа внесенных в базу по внутренней рекомендации.

 

Прочее

Расходы на подбор персонала - отчет, отражающий перечень платежей, по которым в указанном периоде (Дата начала) начато оказание услуг.

Исполнение бюджета - отчет, отражающий поквартальный баланс исполнения бюджета за указанный период.
