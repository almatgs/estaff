Ввод результатов прохождения обучения

По окончанию учебной программы преподаватель может отмечать результат прохождения каждым кандидатом обучения.

  1. откройте карточку учебной группы в разделе Прочее-Учебные группы
  2. нажмите кнопку Пометить как завершенное
  3. отметьте результат прохождения обучения для каждого из кандидатов путем соответствующего выбора (Пройдено, Не пройдено)

Если необходимо зафиксировать результат прохождения обучения до окончания учебной программы, в личном деле кандидата нажмите кнопку Результат под записью Обучение и отметьте итог прохождения обучения путем выбора одной из галочек (Пройдено, Не пройдено).
