Поиск резюме в Интернет

Программа E-Staff Рекрутер позволяет осуществлять поиск резюме сразу по нескольким работным сайтам в Интернет при помощи одного запроса и переносить найденные резюме во внутреннюю базу.

Для доступа к сайтам используются Ваши регистрационные данные, указанные в разделе Персональные настройки->Интернет-модуль->Учетные записи.

  • Для осуществления поиска резюме в интернет через E-Staff выберите в главном меню программы команду Сервис->Поиск в Интернете->Поиск резюме в Интернет (или нажмите кнопку поиска на панели инструментов).

Откроется окно поиска резюме в Интернет:

  • На вкладке Параметры поиска отметьте сайты, на которых планируется производить поиск, и задайте необходимые критерии (как минимум, ключевые слова и регион).

Поиск по ключевым словам. В критериях поиска по ключевым словам можно задать требуемую логику, указывая необходимые условия (Все слова, Любое из слов, Точная фраза, Не встречаются) и область поиска. Кроме того, таких критериев может быть задано несколько - формирование следующей строки списка критериев производится с помощью кнопки + «Добавить условие», расположенной в конце предыдущей строки.

Поиск по регионам. Возможен поиск как в одном, так и в нескольких регионах одновременно, указывая их в поле Регион. Несколько регионов выбираются в списке с использованием клавиш Shift или Ctrl. Поиск с учетом готовности переезда в выбранный регион включается опцией Включая готовых к переезду сюда.

Для поиска с учетом наличия фотографии в резюме необходимо отметить флажок Только резюме с фотографией.

Поиск с учетом владения иностранными языками. В поле Знание языков можно выбрать язык и уровень владения. Кроме того, таких критериев может быть задано несколько - формирование следующей строки списка критериев производится с помощью кнопки + «Добавить язык», расположенной в конце предыдущей строки.

Для поиска на сайте с учетом специальности (рубрики) необходимо выбрать ее в поле Рубрика. Возможен выбор сразу нескольких элементов, удерживая клавишу Ctrl..

  • Нажмите кнопку Найти. Сбросить сразу все заданные ранее критерии поиска можно с помощью кнопки «Очистить форму» на панели инструментов.

Запустится поиск по выбранным сайтам. При необходимости процесс поиска можно прервать кнопкой «Остановить загрузку» (в виде красного квадрата). По окончании поиска на вкладке Результаты будет выдан список найденных резюме.

В верхней половине окна на вкладке Результаты перечисляются заданные критерии поиска, справа – статистика по найденным резюме для каждого сайта, на которых производился поиск:

  • общее количество найденных резюме;
  • количество резюме, загруженных в интернет-модуль;
  • количество нерассмотренных резюме.

Ниже располагается список найденных резюме. Значки ранее просмотренных Вами резюме будет серого цвета, а добавленные в базу резюме – отмечены знаком «+» в столбце Есть в базе.

  • Для просмотра найденного резюме выделите его в списке.

В нижней половине окна в области предварительного просмотра появится содержимое резюме соискателя в стандартизированном виде.

В правом нижнем углу окна расположена область с кнопками для добавления выбранного резюме в базу и просмотра ключевой информации по кандидату, резюме которого было сохранено в базу ранее (для быстрого доступа к его карточке можно использовать ссылку, отображаемую в виде ФИО).

  • Для переноса выбранного резюме в Вашу базу используется кнопка Поместить в базу. Если Вы сразу же определились, на какую вакансию подходит просматриваемый соискатель, перенос резюме в базу можно выполнить параллельно с отбором на эту вакансию, с помощью команды Поместить в базу и отобрать на вакансию из выпадающего меню рядом расположенной кнопки «Все действия». В этом случае откроется окно диалога, предлагающее выбрать вакансию из списка активных. Чтобы в дальнейшем при переносе резюме других соискателей не пришлось каждый раз искать одну и ту же вакансию, можно использовать команду Поместить в базу на текущую вакансию, один раз указав эту вакансию командой Выбрать текущую вакансию.

Откроется карточка кандидата, в которой будут заполнены поля, которые программа смогла распознать. В нижней части экрана будет расположено резюме кандидата в исходном виде.

Перед добавлением резюме программа предварительно выполнит проверку на наличие возможных дубликатов в базе. Если дубликаты будут найдены, программа выдаст соответствующее предупреждение.

  • Для сохранения карточки кандидата в базе нажмите кнопку Сохранить и закрыть.
