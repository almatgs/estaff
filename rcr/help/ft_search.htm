Поиск по ключевым словам

Все разделы программы содержат фильтр Поиск по тексту. В данном поле можно указать одно либо несколько ключевых слов, которые должны присутствовать внутри карточки.

В случае, если слова перечислены через пробел, требуется наличие каждого из этих слов (условие "И")

В случае, если слова перечислены через символ |, требуется наличие хотя бы одного из указанных слов (условие "ИЛИ")

 

По умолчанию программа осуществляет поиск по началу указанных слов, например, секретар делопроизводств. Для осуществления поиска целыми словами, необходимо заключить слова в кавычки, например, "секретарь" "делопроизводство".

Для поиска по произвольным фрагментам слов необходимо использовать символ * для обозначения несущественного фрагмента, например, *net *studio

 

Длина поискового слова должна быть не менее 2-х символов. Строчные и заглавные буквы при поиске не различаются.
