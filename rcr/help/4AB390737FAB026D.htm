Собственные справочники

Созданные в программе собственные справочники, так же как и стандартные, можно будет в дальнейшем привязывать к дополнительным полям.

Создание собственных справочников производится в разделе Администрирование –> Справочники –> Прочие –> Собственные справочники с помощью кнопки Новый элемент в правом верхнем углу.

  • нажмите кнопку Новый элемент данных, появится форма с настройками справочника.
  • введите Наименование создаваемого справочника.
  • поле Код заполняется автоматически, его значение изменять не нужно.
  • укажите тип ключевого поля справочника, определяющий, какого рода информация будет храниться в данном справочнике: Числовой (информация будет представлена числовыми значениями), или Строковый (информация будет представлена в виде текста).
  • если справочник подразумевает некоторую структуру, то установите флажок Иерархический справочник.
  • для включения возможности визуального выделения отдельных элементов справочника необходимо установить флажки Использовать цвет текста и/или Использовать цвет фона.

Созданный справочник появится в разделе Администрирование –> Справочники –> Прочие только после перезапуска сервера E-Staff (в сетевой версии) и последующего запуска программы.

Теперь этот справочник можно наполнить конкретными значениями:

  • выберите его в списке и с помощью кнопки Новый элементсоздайте карточкузначенийсправочника.
  • введите Наименование создаваемого значения справочника.
  • если справочник иерархический, то в поле Родительский элемент укажите головной элемент справочника. Соответственно для элементов верхнего уровня это поле заполнять не нужно.
  • если справочник поддерживает визуальное выделение элементов, то кнопкой [...] возле полей Цвет текста и/или Цвет фона выберите желаемый цвет.
