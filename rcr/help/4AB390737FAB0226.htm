Шаблоны писем

Программа E-Staff позволяет отправлять письма отобранным кандидатам и заказчикам по стандартным шаблонам. Для отправки писем по шаблонам Вы можете использовать кнопки Письмо кандидату и Письмо заказчику в карточке отобранного кандидата. Вместе с программой поставляются следующие стандартные шаблоны:

·         Уведомление кандидата о включении в базу данных

·         Приглашение кандидата на интервью

·         Уведомление кандидата об отказе (прекращении набора на вакансию)

·         Резюме кандидата для заказчика

 При необходимости Вы можете редактировать существующие шаблоны и добавлять собственные. Для этого используется раздел Администрирование:Справочники:Шаблоны писем.

·         Для добавления нового шаблоны нажмите кнопку Новый элемент.

В нижней части раздела появятся поля, соответствующие новому шаблону. Укажите тип шаблона (шаблон для кандидата или шаблон для заказчика), наименование шаблона (Вы будете видеть его в списке при выборе шаблона для отправки), тему и текст письма. В теле письма по аналогии со стандартными шаблонами, вы можете использовать стандартные вставки (ФИО, вакансия, подпись), на место которых программа будет подставлять фактические значения.

Поскольку в многопользовательской версии программы все пользователи используют общие шаблоны, каждому пользователю необходимо указать шаблон своей подписи в разделе Отправка почты пользовательских настроек (пункт Настройки пользователя... меню Сервис).
