Наборы тестов

Наборы тестов предназначены для возможности автоматического назначения кандидату заданного набора тестов на этапе тестирования, в зависимости от вакансии, на которую он рассматривается.

Создание наборов тестов производится в разделе Администрирование –> Справочники –> Интеграция–> Наборы тестов с помощью кнопки Новый элемент в правом верхнем углу.

  • введите Наименование создаваемого набора тестов.
  • нажмите кнопку Добавить тесты, появится окно со списком тестов. Если список пуст, то нажмите кнопку Загрузить справочник, при этом из внешней системы тестирования будет загружен последний актуальный справочник тестов.
  • выделите в списке требуемые для создаваемого набора тесты и нажмите ОК.
