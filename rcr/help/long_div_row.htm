Рекомендации по настройке системы для работы со штатной структурой в крупных (холдинговых) компаниях

Крупные (холдинговые) компании в силу своего масштаба часто имеют весьма сложную иерархическую организационную структуру. В этом случае полная цепочка подразделений может оказаться довольно длинной, например:

ООО «Промышленный центр КРАБ» -> Дивизион продукции производственного назначения -> Отдел разработки -> Группа снабжения -> Транспортный сектор

При таком размере цепочки в отведенном для нее поле в карточках или списках мы не увидим целевого подразделения (в приведенном примере – подразделения «Транспортный сектор»), а увидим только начало цепочки. Существует два дополняющих друг друга варианта решения данной проблемы: 

  • Использование сокращенного наименования подразделений (по умолчанию). Поле Сокращенное наименованиезаполняется в карточке структурного подразделения.

Для нашего примера результат станет примерно следующим:

ООО «ПЦ КРАБ» -> Дивизион ППН -> Отдел разработки -> ГрСнаб -> ТранспСектор

Управление данной функцией осуществляется в разделе Администрирование->Общие настройки->Штатная структура опцией Использовать сокращенное наименование подразделения, если оно указано, при показе полной цепочки подразделений. 

  • Переключение режима отображения иерархической цепочки подразделений на обратный.

Для нашего примера результат станет следующим:

Транспортный сектор <- Группа снабжения <- Отдел разработки <- Дивизион продукции производственного назначения <- ООО «Промышленный центр КРАБ»

А при использовании обоих вариантов:

ТранспСектор <- ГрСнаб <- Отдел разработки <- Дивизион ППН <- ООО «ПЦ КРАБ»

Управление данной функцией осуществляется в разделе Администрирование->Сервис->Объекты->Структурное подразделение опцией Порядок отображения наименования в иерархии.
