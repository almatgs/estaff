Добавление вакансии

Как правило, вакансия заводится пользователем E-Staff (рекрутером) вручную.

  • Для добавления вакансии вручную нажмите кнопку Новая вакансия на панели инструментов. Кроме того, это возможно делать непосредственно из карточки сотрудника/контактного лица с помощью кнопки «Новая вакансия» на панели инструментов. В этом случае поля Структурное подразделение (Клиент - для кадровых агентств) и Контактные лица заполнятся автоматически.

Откроется пустая карточка новой вакансии. По умолчанию новая вакансия будет иметь статус В работе.

Как правило, обязательно заполняются поля Наименование, Структурное подразделение (для кадровых служб компаний) или Клиент (для кадровых агентств).

При выборе структурного подразделения (для кадровых служб компаний) будет показан список действующих подразделений, а затем программа автоматически заполнит список контактных лиц руководителем данного подразделения либо другим сотрудником (если он единственный сотрудник этого подразделения).

При выборе клиента (для кадровых агентств) программа по умолчанию автоматически заполнит список контактных лиц ключевым лицом (у которого включен флажок Основное контактное лицо) данной организации-клиента. Если основного контактного лица не окажется или связь с ним была утеряна, список контактных лиц будет сформирован из состава оставшихся активных лиц организации-клиента.

Автоматическую загрузку списка контактных лиц при выборе клиента при необходимости можно отключить в разделе Администрирование->Общие настройки->Работа с вакансиями, сняв флажок При создании вакансии автоматически загружать список контактных лиц из карточки клиента.

Кроме того, Вы можете изменить сформированный список контактных лиц вручную: добавить запись, нажав справа кнопку Выбрать контактное лицо из списка или удалить, нажав на выбранной позиции списка правой кнопкой и выбрав в контекстном меню Удалить.

По умолчанию при заведении вакансии создается одиночная вакансия (опция Одиночная).

Опция На несколько человек позволяет вести подбор нескольких человек на однотипную вакансию. На каждого требуемого человека будет заведена т.н. детализированная вакансия со своим набором параметров (дата открытия, основание появления, приоритет, срок поиска, рекрутер, финальный кандидат, ставка), хранящаяся в карточке основной вакансии (раздел Детализация). В программе предусмотрена возможность массовой смены детализаций с помощью команды Изменить статус контекстного меню, вызываемого правой кнопкой мыши на группе выбранных карточек детализаций.

Опция Массовая (поток) позволяет вести подбор неограниченного количества человек на поточную вакансию, которая, как правило, никогда не закрывается.

  • Заполнив требуемые поля в карточке вакансии, нажмите Сохранить и закрыть.

 

При необходимости E-Staff может быть интегрирован с корпоративным порталом компании таким образом, что руководитель структурного подразделения сможет подавать заявки на вакансии через Web-интерфейс портала, после чего они будут автоматически подгружаться в E-Staff в качестве новых неутвержденных вакансий (со статусом Новая заявка).
