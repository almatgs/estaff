Интеграция с календарем Microsoft Outlook

Для настройки выгрузки запланированных событий (как правило, Интервью и Интервью с заказчиком) в календарь Microsoft Outlook и (или) их загрузки из него, необходимо выполнить следующее:

  • Нажмите Сервис->Персональные настройки
  • Откройте пункт Календарь. Укажите MS Outlook в качестве внешнего календаря.
  • Откройте пункт Модули к внешним программам. Напротив записи Microsoft Outlook нажмите кнопку Установить.

После этого запланированное интервью будет автоматически выгружаться в календарь MS Outlook. При изменении даты/времени этого события в E-Staff в календаре MS Outlook оно тоже передвинется.

  • Если необходимо, чтобы событие Интервью с заказчиком выгружалось в MS Outlook вместе с прикрепленным резюме кандидата, то можно настроить выгрузку события по шаблону. В этом случае в Outlook будет выгружена карточка собрания с прикрепленным файлом резюме. Для этого необходимо создать специальный шаблон письма для выгрузки (подробно о работе с шаблонами - в разделе Шаблоны писем). В создаваемом шаблоне укажите Тип получателя - Запись во внешнем календаре и заполните поля Тема письма и Текст письма. Информация, набранная в теме письма, будет отображаться в карточке собрания MS Outlook в качестве его названия. При желании, как в теме, так и в тексте письма можно использовать различные вычисляемые элементы. Напротив надписи Вложенные файлы из карточки нажмите кнопку «+» (Добавить), укажите имя файла (рекомендуется вставить вычисляемый элемент Кандидат: Фамилия, Имя, Отчество), тип приложения - Резюме, исходный объект - Кандидат. Затем в разделе Администрирование –> Справочники –> Общие –> Типы событий откройте карточку события Интервью с заказчиком и на закладке Доп. сведения в качестве шаблона для формирования карточки собрания во внешнем календаре выберите созданный ранее шаблон.

 

  • Для загрузки запланированных событий из календаря MS Outlook во внутренний календарь E-Staff используется фоновая задача Загрузка из внешнего календаря. Настройка  этой фоновой задачи производится в разделе Администрирование -> Справочники –> Прочие -> Локальные фоновые задачи.

В ручном режиме данная задача запускается из главного меню программы командой Сервис -> Ручной запуск фоновой задачи -> Загрузка из внешнего календаря.

Для настройки загрузки по расписанию (ежедневно или через иной промежуток времени) в полях Час и Минута укажите требуемые значения.

При импорте мероприятий из календаря Microsoft Outlook мероприятия, помеченные как личные (частные), будут импортироваться без темы и описания.

При изменении даты/времени запланированного события в календаре MS Outlook в календаре E-Staff оно тоже передвинется (после отработки фоновой задачи Загрузка из внешнего календаря в ручном или автоматическом режиме).
