Общие принципы работы с разделами программы

Для доступа к карточкам различных типов используется дерево разделов, расположенное в левой части главного окна программы. Все разделы программы, перечисленные в дереве, построены по единому принципу. При необходимости, любой из разделов программы можно открыть в отдельном окне (Ctrl+N на клавиатуре или командой в меню Вид->Открыть раздел в новом окне). Разделы, включающие в себя подразделы, можно сворачивать. Программа запоминает состояние каждого раздела, поэтому при последующем открытии программы они будут иметь выбранный ранее вид.

В правой части экрана отображается список объектов, соответствующих выбранному разделу (подразделения, вакансии, кандидаты и т.д.).

  • Для отбора данных в списке по различным критериям задайте нужный набор значений фильтров в верхней части экрана над списком. Вслучае выбора значения фильтра, содержащего набор элементов, отбор выполняется сразу же после выбора значения. А в случае использования фильтра с вводом текста, для применении фильтра необходимо нажать кнопку Найти, или, находясь в текстовом поле, нажать клавишу Enter.

Набор динамических фильтров для текущего раздела можно менять с помощью команды Вид –> Фильтры главного меню. С помощью команд Переместить выше/Переместить ниже контекстного меню в области Используемые фильтры, Вы можете поменять расположение фильтров.

По умолчанию изменение набора фильтров происходит только для данного рабочего места (режим Настроить только для меня). Для централизованного внесения изменений администратор приложения должен выбрать режим Настроить для всех пользователей.

В добавленных фильтрах, ссылающихся на справочник (например, Регион, Специальность), можно выбирать сразу нескольких значений, поочередно указывая их, удерживая клавишу Ctrl. Это удобно делать в окне выбора элементов, вызываемого командой Выбрать из списка (правой кнопкой мыши на фильтре).

·  Для сброса отдельного фильтра с выпадающим списком значений необходимо правой кнопкой мыши нажать на этом фильтре и выбрать команду Очистить поле. Для сброса всех используемых фильтров необходимо правой кнопкой мыши нажать в свободную область вокруг фильтров и выбрать Очистить форму.

Набор столбцов и их последовательность для текущего раздела можно менять с помощью команды Вид –> Столбцы главного меню аналогично смене динамических фильтров.

·  Для изменения порядка сортировки объектов щелкните мышкой по заголовку колонки, по которой должна производиться сортировка.

·  Для изменения порядка сортировки на обратный повторно щелкните мышкой по заголовку колонки.

·  Для открытия карточки объекта дважды щелкните мышкой на соответствующую строку в списке.

·  Для удаления карточки объекта выделите соответствующую строку в списке, затем нажмите клавишу Delete.

Также для открытия или удаления объекта можно использовать контекстное меню, выпадающее при нажатии правой кнопки мыши на строке в списке.

 

Над списком отображается набор полей, по которым можно осуществлять фильтрацию объектов в списке. Первым всегда идет полей Поиск по тексту, позволяющее указать любое ключевое слово (или фрагмент) слова, которое должно содержаться внутри карточки. Для получения более подробной информации см. раздел Поиск по ключевым словам.
