Справочники

Раздел Администрирование->Справочники содержит справочники, используемые внутри программы. Общие для всех объектов справочники вынесены в подраздел Общие, остальные же, используемые только в тех или иных объектах (контрагенты, вакансии, кандидаты) - в отдельные соответствующие подразделы.

Справочники, в зависимости от назначения, могут быть одноуровневыми (например, Бренды, Статьи затрат), либо иерархическими.

Выбор того или иного элемента справочника при работе с программой производится из выпадающего списка или в диалоговом окне с функцией поиска. Способ выбора элемента каждого из справочников задается в разделе Администрирование->Справочники в окне его свойств, вызываемом по правой кнопке мыши (Свойства справочника) в свободной области возле поля Поиск по тексту. Там же можно включить режим сортировки элементов справочника по их наименованию.

  • Для добавления нового элемента справочника нажмите кнопку Новый элемент. Для иерархических справочников в поле Родительский элемент укажите элемент вышестоящего уровня.

Измененные пользователем стандартные элементы справочников помечаются в графе Тип элемента символом (*).  Для сброса измененного элемента справочника в первоначальное состояние, заданное в дистрибутиве программы, щелкните на этом элементе правой кнопкой мыши, выберите пункт Сбросить в первоначальное состояние и следуйте далее указаниям программы.

  • Для добавления нового элемента справочника на основе существующего, в карточке имеющегося элемента нажмите кнопку Файл->Создать дубликат карточки.
  • Для удаления элемента справочника выделите его в списке и, нажав правой кнопкой мыши, выберите пункт Удалить, или на клавиатуре нажмите клавишу Delete.

Примечание. Стандартные элементы отдельных справочников невозможно удалить, т.к. они могут быть связаны с работой сразу нескольких модулей программы либо с протоколами обмена при интеграции с внешними системами или сайтами.
