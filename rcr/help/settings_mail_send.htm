Отправка почты

Раздел Отправка почты служит для настройки отправки писем кандидатам и заказчикам по шаблонам. Отправка писем возможна либо через Вашу почтовую программу либо напрямую через SMTP-сервер. В обоих случаях Вам потребуется произвести некоторые настройки E-Staff Рекрутер.

Пункт Метод подготовки единичных писем используется для выбора способа взаимодействия с почтовой программой или выбора отправки через SMTP-сервер.

Как правило, для большинства почтовых программ будет работать метод Simple MAPI, выбранный по умолчанию. В этом случае для нормального функционирования Ваша почтовая программа должна быть выбрана в операционной системе Windows как используемая по умолчанию. Для этого:

·  Запустите программу Internet Explorer и выберите команду Свойства браузера из меню Сервис.

·  В открывшемся окне диалога выберите закладку Программы.

·  Нажмите кнопку Задать программы, далее в открывшемся окне – Настройка доступа программ и умолчаний.

·  В открывшемся списке выберите Вашу почтовую программу, установите флажок Сделать доступной эту программу и нажмите OK.

Данный метод нормально функционирует при использовании почтовых программ Outlook Express, Microsoft Outlook, The Bat!.

Если при использовании почтовой программы Microsoft Outlook формирование письма для отправки не происходит - выберите метод подготовки Microsoft Outlook automation и в разделе Модули к внешним программам напротив записи Microsoft Outlook нажмите кнопку Установить (если модуль не был установлен ранее).

При использовании почтовой программы Lotus Notes можно использовать как метод Simple MAPI, так и метод Lotus Notes.

При этом необходимо, чтобы Lotus Notes был установлен из дистрибутива, и после его установки не устанавливалось других почтовых программ. Если отправка писем не работает, переустановите Lotus Notes на Вашем компьютере.

Для проверки взаимодействия E-Staff с Вашей почтовой программой выбранным методом нажмите кнопку Подготовить тестовое сообщение. В используемой почтовой программе должно открыться сообщение с текстом Test.

Метод подготовки через SMTP может использоваться при отсутствии на рабочем месте почтовой программы. В этом случае каждое письмо создается и предварительно открывается непосредственно в E-Staff. Использование функции отправки через SMTP возможно с помощью общих настроек SMTP-сервера компании (ссылка Общие настройки SMTP для быстрого перехода к настройкам) или персональных (например, используя настройки исходящей почты и регистрационные данные своего бесплатного электронного почтового ящика). В полях Адрес отправителя и Имя отправителя введите электронный адрес (с которого будут отправляться письма) и имя соответственно (при использовании общих настроек SMTP письма в этом случае будут отправляться с электронного адреса пользователя, а не с общего корпоративного адреса, указанного в настройках автоматической отправки почты). Для проверки правильности введенных данных в поле Тестовый E-mail укажите свой электронный адрес и нажмите кнопку Отправить тестовое сообщение. При корректно введенных данных на указанный почтовый адрес придет сообщение с текстом Test.

Для правильной настройки может потребоваться помощь Вашего системного администратора. В отдельных случаях настройку сможет произвести только системный администратор.
