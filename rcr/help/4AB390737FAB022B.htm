Заведение нового пользователя

Заводить пользователя может либо администратор системы, либо другой пользователь с правами "Администратор приложения".

  • Для заведение нового пользователя сетевой версии программы перейдите в раздел Администрирование->Доступ->Пользователи и нажмите кнопку Новый пользователь.

    Откроется карточка нового пользователя:

  • Укажите имя пользователя и пароль и укажите уровень доступа пользователя.

    Флажок Дублировать внутренние уведомления программы на E-Mail означает включение функции автоматического дублирования внутренних уведомлений программы на E-Mail пользователя с помощью функции автоматической отправки почты, без предварительного открытия уведомлений в Вашей почтовой программе. E-Mail пользователя должен быть вписан в соответствующей ему карточке сотрудника, указываемую в поле Сотрудник. Настройка функции автоматической отправки почты осуществляется в разделе Администрирование–>Общие настройки–>Автоматическая отправка почты. Для правильной настройки может потребоваться помощь Вашего системного администратора. В отдельных случаях настройку сможет произвести только системный администратор.

    При закрытии карточки программа предложит сохранить данные о пользователе в базе. Выберите ответ Да. Также Вы можете сохранить данные о пользователе заранее, выбрав команду Сохранить из меню Файл (или нажав Ctrl+S).
