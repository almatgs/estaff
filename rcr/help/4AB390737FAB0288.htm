Списки контактов

 

  • Для добавления нового списка, в меню Файл выберите команду Новый список контактов.

Откроется карточка списка контактов, имеющая следующую структуру:

Общие сведения:

Наименование: Наименование списка контактов.

Контактные лица: Перечень добавленных в список контактных лиц с отображением ключевой информации по каждому из них.

Комментарий: Произвольная дополнительная информация.

 

Левая панель:

В архиве: Флажок, обозначающий, что с данным списком прекращена работа. Объект базы становится неактивным и обозначается в списке картинкой серого цвета.

 

  • Для добавления элемента в список нажмите на панели инструментов кнопку Добавить контактные лица. Кроме того, это возможно делать непосредственно из карточки сотрудника/контактного лица с помощью команды Действия->Добавить в список контактов, либо с помощью такой же команды в контекстном меню соответствующего списка сотрудников/контактных лиц.

Откроется диалоговое окно со списком физических лиц (по умолчанию - из числа контактных лиц и сотрудников) для выбора.

  • Выберите одно или несколько лиц и нажмите ОК. Несколько лиц выбираются в списке с использованием клавиш Shift или Ctrl.

Перечень добавленных в список контактных лиц с отображением ключевой информации по каждому из них появится в поле Контактные лица.

  • Для сохранения созданного списка в базу нажмите кнопку Сохранить и закрыть.

Созданный список появится в основном дереве программы в разделе Контрагенты-Списки контактов (Контактные лица-Списки контактов - для рекрутинговых агентств).
