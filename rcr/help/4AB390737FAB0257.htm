Интеграция с внешней системой кадрового учета (обмен файлами)

Интеграция с внешней системой кадрового учета используется для автоматической загрузки данных о структурных подразделениях, штатных должностях, сотрудниках и вакансиях компании.

Для загрузки данных из внешней системы кадрового учета в E-Staff, они должны быть предварительно выгружены в специально созданную директорию (исходную директорию для импорта) в виде файлов формата CSV, имена которых должны начинаться со слов:

  • division (данные по штатной структуре),
  • position (данные по штатным должностям),
  • person (данные по сотрудникам),
  • vacanc (данные по вакансиям).

После загрузки и обработки этих файлов, E-Staff переместит их в директорию для обработанных файлов. Для контроля корректного завершения обработки рекомендуется настроить выгрузку из внешней системы таким образом, чтобы указанные выше имена файлов заканчивались датой выгрузки, чтобы в директории для обработанных файлов по наличию файлов с той или иной датой было понятно, когда обработка прошла успешно.

Ключевым параметром требуемого формата обмена является идентификатор id. При первой выгрузке из внешней системы каждому объекту присваивается определенный id, который при последующих выгрузках должен оставаться неизменным.

 

Настройка параметров взаимодействия с внешней системой кадрового учета

  • Перейдите в E-Staff в раздел Администрирование -> Общие настройки -> Внешняя система кадрового учета.
  • В настройках выберите интерфейс Обмен файлами, Формат файла – CSV, кнопками Просмотр укажите исходную директорию для импорта и директорию для обработанных файлов.

Если вторая строка в CSV-файле структурных подразделений содержит наименование самой организации, а не первого подразделения, то в настройках установите флажок Не загружать корневое подразделение, соответствующее единственной организации.

 

В ручном режиме запуск загрузки данных производится выбором соответствующей команды в главном меню программы: Сервис -> Ручной запуск фоновой задачи. Для загрузки данных о структурных подразделениях, штатных должностях и сотрудниках служит команда Загрузка штатной структуры и сотрудников из внешней системы, а для загрузки данных о вакансиях - Загрузка вакансий из внешней системы.

Для загрузки данных из внешней системы по расписанию (ежедневно или через иной промежуток времени) перейдите в раздел Администрирование -> Сервис -> Фоновые задачи и в настройках соответствующей фоновой задачи укажите требуемые значения.

При значительной длине полной иерархической цепочки подразделений, она может не помещаться в отведенное для нее поле в карточках и списках, в результате чего становится не видно наименования целевого подразделения. Но существует два дополняющих друг друга варианта решения данной проблемы. См. Рекомендации по настройке системы для работы со штатной структурой в крупных (холдинговых) компаниях.

Если на момент загрузки данных из внешней системы кадрового учета карточки подразделений, штатных должностей, сотрудников и вакансий были заведены ранее вручную, то в базе появятся дубликаты этих карточек. Инструкция по их устранению находится в разделе Удаление дубликатов после загрузки данных из внешней системы.

В случае использования интеграции с внешней системой для загрузки данных о сотрудниках компании в E-Staff, можно настроить формирование автоматической связки принятого на работу кандидата и соответствующего сотрудника, позволяющей быстро переключаться между ними, а также автоматически менять статус связанного кандидата в случае увольнения сотрудника. См. Автоматическое формирование связки кандидата и сотрудника.
