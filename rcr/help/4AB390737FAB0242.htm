Регионы

Справочник Регионы содержит список регионов (областные центры, области, районные центры нескольких стран), используемых в системе.

Данный справочник имеет настраиваемый механизм загрузки. Для заполнения справочника необходимыми для работы регионами выполните следующие действия:

  • нажмите кнопку Загрузить. Откроется окно загрузки справочника регионов.
  • выберите страну и укажите тип подгружаемых регионов переключением режима Глубина загрузки. В нижней части окна появится соответствующий список регионов (предварительно выделенных).
  • для заполнения справочника полным списком регионов согласно указанным настройкам нажмите ОК.
  • для заполнения справочника только отдельными регионами выделите их в списке и нажмите ОК.
