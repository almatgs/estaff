Шаблоны вакансий

Справочник Шаблоны вакансий содержит список созданных пользователем шаблонов вакансий для возможности быстрого заполнения раздела Объявление в Интернет карточки вакансии по заранее подготовленному шаблону. Для выбора шаблона используется поле Шаблон для публикации в разделе Доп. сведения карточки вакансии.

Создание собственных шаблонов вакансий производится в разделе Администрирование –> Справочники –> Вакансии –> Шаблоны вакансий  с помощью кнопки Новый элемент в правом верхнем углу.

Карточка шаблона вакансии имеет следующую структуру:

 

Раздел Общие сведения:

Наименование: Наименование шаблона вакансии.

Специальность: Классификатор объектов по специальностям (значение из справочника).

Регион: регион будущего места работы (значение из справочника). Используется в том случае, если компания ведет детальность в нескольких регионах.

 

Раздел Объявление в Интернет:

Вкладки Общее, Описание, Раздельное описание используются для заполнения информацией, необходимой для размещения. Поля вкладки Раздельное описание используются для заполнения при размещении вакансии на сайте superjob.ru и становятся доступными при установленном флажке superjob.ru.
