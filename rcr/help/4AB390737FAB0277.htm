Источники кандидатов

Справочник Источники кандидатов содержит список источников, используемых в системе для быстрого указания в карточке кандидата источника поступления (сайт, кадровое агентство, рекомендация).

Основными источниками поступления кандидатов (они уже присутствуют в программе), как правило, являются несколько известных работных сайтов (job.ru, rabota.ru, hh.ru, hh.ua, hh.kz, rabota.mail.ru, zarplata.ru, superjob.ru). При появлении в Вашей работе других работных сайтов или прочих дополнительных источников поступления кандидатов, таких как кадровое агентство, СМИ, рекомендуем их добавить и произвести их дополнительную настройку в разделе Администрирование–>Справочники–>Кандидаты–>Источники кандидатов:

  • если новый источник поступления является организацией, то для его автоматического добавления в данный справочник в карточке контрагента можно использовать флажок Источник поступления кандидатов. В остальных случаях для создания нового источника поступления нажмите кнопку Новый элемент в правом верхнем углу.
  • в поле Наименование введите имя добавляемого источника, укажите (при необходимости) родительский элемент.
  • в поле Ключевые слова (почтовые домены) можно ввести, например, домены сайта (в т.ч. почтовые – например, домен почтового сервера кадрового агентства) или ключевое слово. Если резюме будет помещаться в базу с сайтов, имеющих указанные домены, или текст самого резюме будет содержать название этих сайтов либо ключевое слово, то поле Источник в карточке кандидата будет заполняться автоматически. В общем случае будет возможность указать требуемый источник поступления вручную двойным щелчком мыши по полю Источник.
  • если добавляемый источник является организацией, то привяжите его в поле Карточка контрагента к соответствующей заведенной ранее карточке контрагента.
  • если добавляемый источник импорта является сайтом, то установите флажок Сайт.
  • сохраните карточку источника кнопкой Сохранить и закрыть.
