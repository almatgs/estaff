Перестройка индексных файлов

В сетевой версии E-Staff

На компьютере, где установлен сервер E-Staff, необходимо выполнить следующее:

  • Запустите E-Staff Server из меню Пуск -> Программы (Start -> Programs) и выключите сервер E-Staff.
  • В меню выберите Сервис -> Удалить индексные файлы.
  • Включите сервер.

 

В однопользовательской версии E-Staff

  • Закройте E-Staff Рекрутер.
  • Удалите папку ft из папки data_rcr в установочной директории программы (по умолчанию - C:\Program Files\EStaff).
  • Запустите E-Staff Рекрутер.

 

Дополнительная информация:

  • Перестройка индексных файлов происходит в фоновом режиме. В программе в это время можно работать, но поиск по ключевым словам работать не будет.
  • Окончание перестройки можно определить по исчезновению файла rebuild в папке ft и исчезновению всех файлов из папки queue.
