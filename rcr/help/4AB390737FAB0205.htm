Аппаратно-программные требования

  • Рабочее место
    • Операционная система Microsoft Windows XP, Server 2003, Vista, Server 2008, Windows 7, Windows 8
    • CPU от 1.2 GHz
    • 1 Gb оперативной памяти.
    • Microsoft Office 2003 или более поздняя версия (для импорта резюме из документов Microsoft Word или интеграции с календарем Microsoft Outlook), или OpenOffice.org 2.3 или более поздняя версия (для импорта резюме из документов OpenOffice.org).
  • Сервер (для сетевой версии, до 10 рабочих мест)
    • Операционная система Microsoft Windows Server 2003, Server 2008, Server 2012 
    • CPU от 2.2 GHz
    • 4 Gb оперативной памяти.
  • Сервер (для сетевой версии, 10 и более рабочих мест)
    • Операционная система Microsoft Windows Server 2003, Server 2008, Server 2012 
    • CPU от 2.2 GHz x 2
    • 8 Gb оперативной памяти.
