Дополнительные поля

Вы можете расширять стандартные карточки кандидата, вакансии, структурного подразделения, организации, заказчика и события дополнительными полями, используемыми в Вашей компании. Для этого используется раздел Администрирование->Сервис->Дополнительные поля.

Дополнительные поля могут быть добавлены в карточку выбранного типа объекта на вкладку Общие сведения (по умолчанию карточки всех объектов открываются на этой вкладке) или могут быть вынесены на отдельную созданную вкладку.

У пользователя, который будет создавать дополнительные поля, должны быть права на редактирование общих настроек.

 

Для расширения стандартных карточек дополнительными полями откройте соответствующий тип объекта.

Нажмите кнопку Новый элемент данных, появится форма с настройками дополнительного поля.

Введите наименование и тип нового элемента.

Поле Идентификатор заполняется автоматически, его значение изменять не нужно. Поле Тип данных определяет, какого типа информация будет храниться в данном поле.

Строка – если информация, которую Вы будете вносить в данное поле – текст.

Целое число – если информация, которую Вы будете ввносить будет целым числом.

Действительное число – если информация, которую Вы будете вносить в данное поле, будет числом, и есть вероятность, что данное число может быть отрицательным.

Флаг – поле флагового типа (да/нет в зависимости установлен флажок либо снят).

Дата – если в поле будет вноситься информация в формате даты.

Применительно к типу данных Строка:

  • Флажок Многострочное поле позволяет создать поле высотой более одной строки (параметр Высота).
  • В поле Ссылка на справочник можно указать привязку к любому справочнику программы(Специальности, Регионы и т.д., а такжек созданному собственному справочнику - см. Собственные справочники). Флажок Выбор из нескольких значений включит для пользователя возможность указания сразу нескольких значений выбранного справочника.

Применительно к типу данных Целое число:

  • В поле Ссылка на объект можно указать привязку к списку объектов выбранного типа (Структурное подразделение, Вакансия и т.д.). Флажок Выбор из нескольких значений включит для пользователя возможность указания сразу нескольких значений из списка объектов выбранного типа.

Ширина дополнительного поля регулируется задаваемым числовым значением в процентах или символах в поле Ширина.

Расположением дополнительных полей можно управлять (располагать в строку, добавлять поясняющие надписи, выносить на отдельную вкладку).

Для размещения дополнительных полей в одну строку:

  • создайте управляющий элемент LINE (указатель начала строки), нажав кнопку Новый управляющий элемент и указав тип управляющего элемента Начало строки.
  • создайте управляющий элемент LINE-END (указатель конца строки), нажав кнопку Новый управляющий элемент и указав тип управляющего элемента Конец строки.
  • с помощью команд Переместить выше/Переместить ниже контекстного меню, вызываемого правой кнопкой мыши, расположите управляющие элементы LINE и LINE-END таким образом, чтобы требуемые дополнительные поля находились между ними.

Для визуально-смыслового структурирования дополнительных полей можно добавить поясняющую надпись над ними:

  • создайте управляющий элемент LABEL (указатель надписи), нажав кнопку Новый управляющий элемент и указав тип управляющего элемента Надпись.
  • с помощью команд Переместить выше/Переместить ниже контекстного меню, вызываемого правой кнопкой мыши, расположите управляющий элемент LABELнад темидополнительными полями, над которыми должна располагатьсяпоясняющая надпись.

Для размещения в структуре карточки дополнительных полей на отдельной вкладке:

  • создайте управляющий элемент PAGE (указатель начала раздела), нажав кнопку Новый управляющий элемент и указав тип управляющего элемента Начало раздела.
  • создайте управляющий элемент PAGE-END (указатель конца раздела), нажав кнопку Новый управляющий элемент и указав тип управляющего элемента Конец раздела.
  • с помощью команд Переместить выше/Переместить ниже контекстного меню, вызываемого правой кнопкой мыши, расположите управляющие элементы PAGE и PAGE-END таким образом, чтобы размещаемые дополнительные поляс необходимымиуправляющими элементами находились между ними.

Добавленные Вами поля будут доступны в соответствующих карточках после перезапуска сервера E-Staff (в сетевой версии) и программы.
