Станции метро

Справочник Станции метро содержит список станций метрополитена одного либо нескольких городов.

Для заполнения справочника необходимыми для работы станциями метро выполните следующие действия:

  • нажмите кнопку Загрузить. Откроется окно выбора города, станции метро которого возможно загрузить.
  • для заполнения справочника списком станций метро выберите город и нажмите ОК.
