Шаблоны писем

Программа E-Staff Рекрутер позволяет отправлять письма отобранным кандидатам и заказчикам по стандартным шаблонам. Для отправки писем по шаблонам Вы можете использовать кнопки Начальный email и Резюме у заказчика в карточке отобранного кандидата. Вместе с программой поставляется более десятка стандартных шаблонов писем.

При необходимости Вы можете редактировать существующие шаблоны и добавлять собственные. Для этого используется раздел Администрирование->Справочники->Шаблоны писем.

  • Для добавления нового шаблона нажмите кнопку Новый элемент.
  • В открывшемся окне введите Наименование шаблона (Вы будете видеть его в списке при выборе шаблона для отправки), укажите Тип получателя (шаблон для контактного лица, кандидата, пользователя, фиксированного адресатаили для записи во внешнем календаре), тему и текст письма. Если выбран тип получателя - Пользователь, то необходимо проверить, что в настройках пользователей (раздел Администрирование->Доступ->Пользователи) указан соответствующий сотрудник и в карточке этих сотрудников вписан E-Mail.

В теле письма по аналогии со стандартными шаблонами, вы можете использовать стандартные вычисляемые элементы (ФИО, вакансия, подпись), на место которых программа будет подставлять фактические значения.

Примечание: для корректного формирования сообщений с вычисляемыми элементами необходимо, чтобы на момент создания сообщения в карточке присутствовали данные, которые подставляет тот или иной вычисляемый элемент.

Для добавления файла, наличие которого обязательно при создании сообщения по данному шаблону, напротив надписи Статистические вложенные файлы нажмите кнопку «+» (Добавить), укажите файл и нажмите кнопку Открыть.

Для добавления файла из приложения карточки,  напротив надписи Вложенные файлы из карточки нажмите кнопку «+» (Добавить), укажите имя файла (можно использовать вычисляемые элементы), тип приложения и исходный объект, из которого приложение указанного типа будет подгружаться.   Если это приложение может присутствовать не во всех карточках, то установите флажок Не прикреплять файл в случае его отсутствия внутри карточки.

Поле Скрипт предназначено для ввода программного кода сценария, который будет выполняться при вызове шаблона, для реализации дополнительных функций (разрабатывается индивидуально).

Если требуется чтобы факт отправки сообщения по данному шаблону фиксировался, установите флажок Регистрировать факт отправки письма в виде события и выбрать необходимое событие.

 

Флажок Новая группа в меню позволяет при открытии списка шаблонов из карточки отделить этот шаблон вместе с теми, которые ниже его в общем списке шаблонов, от остальных.

Указав группу в поле Использовать только внутри группы пользователей, можно ограничить доступность текущего шаблона для пользователей пределами указанной группы.

Поскольку в многопользовательской версии программы все пользователи используют общие шаблоны, каждому пользователю необходимо заполнить шаблон своей подписи в разделе Сервис->Персональные настройки (пункт Подпись).
