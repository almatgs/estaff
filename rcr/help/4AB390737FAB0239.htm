Карточка структурного подразделения

Карточка структурного подразделения в программе E-Staff Рекрутер имеет следующую структуру:

Раздел Общие сведения:

Поле Номер содержит внутренний (табельный) подразделения. Номер заполняется программой автоматически, тем не менее Вы можете исправлять его вручную. Если Вас не устраивает предлагаемый программой способ нумерации, Вы можете изменить его в разделе Администрирование->Объекты.

Поле Наименование содержит наименование подразделения, предназначенное для внутреннего использования.

Поле Полное (официальное) наименование содержит наименование подразделения для использования в официальных документах. Управление функцией осуществляется в разделе Администрирование->Общие настройки->Управление полями карточек.

Поле Сокращенное наименование содержит краткое наименование подразделения, предназначенное для сокращения длины отображения полной иерархической цепочки подразделений. Управление функцией осуществляется в разделе Администрирование->Общие настройки->Штатная структура опцией Использовать сокращенное наименование подразделения, если оно указано, при показе полной цепочки подразделений.

Поле Головное подразделение содержит ссылку на головное (родительское) подразделение. Для подразделений верхнего уровня это поле не заполняется.

Поле Дата ввода содержит дату появления подразделения в штатной структуре.

Поле Дата закрытия содержит дату прекращения существования подразделения в штатной структуре. Если данное поле заполнено, и текущая дата больше даты закрытия, подразделения считается закрытым (неактивным) и обозначается в списке серой картинкой.

Поле Организация (опционально) содержит ссылку на организацию, к которому относится данное структурное подразделение (для случая использования режима смешанной штатной структуры).

В поле Сотрудники отображается список сотрудников, относящихся к данному подразделению.

В поле Дочерние подразделения отображается список подразделений, для которого данное подразделение является головным.

 

Раздел Доп. сведения:

Внешний ID: В случае, если подразделение было загружено из корпоративного портала или внешней системы кадрового учета, данное поле будет содержать идентификатор соответствующего объекта во внешней системе.

 

Раздел Вакансии содержит список вакансий по данному подразделению.
