Отправка единичных писем

При помощи E-Staff возможно осуществлять автоматическую подготовку писем по различным шаблонам для дальнейшей отправки кандидатам или контактным лицам по вакансии. Письма могут формироваться автоматически при наступлении определенного события либо это можно делать вручную в карточке физического лица путем нажатия на значок с изображением конверта над полем E-Mail и выбором требуемого шаблона в раскрывшемся списке. Отправка единичных писем возможна двумя способами:

  • Через почтовую программу. В этом случае E-Staff создает и предварительно открывает каждое письмо в Вашей почтовой программе. После этого Вы можете нажать кнопку Отправить, и письмо будет отправлено с сохранением в папке Отправленные. Перед тем, как Вы нажмете кнопку Отправить, Вы можете внести изменения в подготовленный текст письма.

Как правило, использование этой функции не требует дополнительных настроек Вашей почтовой программы. Тем не менее, если при нажатии на кнопки, связанные с отправкой писем, программа не выполняет никаких действий, или выдается системное сообщение об ошибке, следует осуществить необходимые настройки в разделе Отправка почты (Сервис->Персональные настройки->Технические).

  • Через SMTP. Такой вариант подойдет для пользователей, не использующих почтовые программы. В этом случае каждое письмо создается и предварительно открывается непосредственно в E-Staff. Перед тем, как Вы нажмете кнопку Отправить, Вы можете внести изменения в подготовленный текст письма. Выбрать этот вариант отправки можно в разделе Сервис->Персональные настройки->Технические->Отправка почты.

Использование функции отправки через SMTP возможно с помощью общих настроек SMTP-сервера компании (ссылка Общие настройки SMTP для быстрого перехода к настройкам) или персональных (например, используя настройки исходящей почты и регистрационные данные своего бесплатного электронного почтового ящика). Для правильной настройки может потребоваться помощь Вашего системного администратора. В отдельных случаях настройку сможет произвести только системный администратор.

Также Вы можете редактировать шаблоны писем в разделе Шаблоны писем настроек (раздел Администрирование->Справочники->Общие->Шаблоны писем).

Ваша подпись к письму задается в разделе Сервис->Персональные настройки->Общие->Подпись. В случае отправки писем через почтовую программу Microsoft Outlook, Вы можете использовать готовую форматированную подпись из настроек Microsoft Outlook. Для этого в разделе Сервис->Персональные настройки->Технические:

  • Отправка почты - укажите метод подготовки единичных писем Microsoft Outlook automation.
  • Модули к внешним программам - напротив записи Microsoft Outlook нажмите кнопку Установить (если модуль не был установлен ранее).
