Работа с внесенными в базу кандидатами

Список кандидатов базы можно просматривать в разделе Кандидаты либо в подразделе Кандидаты->Поиск кандидатов. Оба раздела поддерживают режим быстрого просмотра выбранного кандидата без открытия его карточки, включаемый правой кнопкой мыши на карточке любого кандидата командой контекстного меню Режим предварительного просмотра.

Для получения информации по соискателям, отобранным на активные вакансии, используется раздел Кандидаты->По вакансиям. После выбора вакансии, над списком кандидатов автоматически сформируются статистические показатели, отражающие текущий прогресс процесса подбора по ней. В левой части выводятся общие показатели – общее количество кандидатов, отобранных на данную вакансию, сколько из них сейчас рассматривается, сколько было отклонено или включено в резерв. Правее выводятся показатели, отражающие количество кандидатов, находящихся на каждом из этапов (в каждом из статусов). Для получения детализации по любому из показателей, достаточно щелкнуть по нему мышью.

Вновь занесенный в базу кандидат имеет статус Новый кандидат или Отклик на вакансию в зависимости от способа его занесения в базу. Личное дело такого кандидата еще либо пусто, либо содержит одну начальную информационную запись (Отклик на вакансию). По мере работы  с этим кандидатом в его личное дело будут заноситься различные записи, последняя из которых будет определять его статус.

Каждый кандидат в определенный момент времени является либо активным ("в работе"), либо архивным (работа по нему на данный момент прекращена). Архивные кандидаты показываются в списках серыми картинками.

Вновь занесенный кандидат является активным до тех пор, пока все вакансии, на которые он рассматривается, не будут закрыты. Если кандидат был отклонен или помещен в резерв до того, как был отобран на какую-либо из вакансий, то он также становится архивным.

Повторный отбор архивного кандидата на вакансию возвращает ему активное состояние.

В программе предусмотрена возможность массовой смены статуса кандидатов с помощью команды Изменить статус контекстного меню, вызываемого правой кнопкой мыши на группе выбранных карточкек кандидатов. В сетевой версии программы возможно ограничить доступность отдельных статусов данной команды для пользователей пределами их групп, указав их в поле Актуально только для группы в разделе Доп. сведения карточки настроек события (Администрирование->Справочники->Общие->Типы событий).
