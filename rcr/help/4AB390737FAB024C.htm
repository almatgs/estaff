Интеграция с календарем Lotus Notes

Для настройки выгрузки запланированных событий (как правило, Интервью и Интервью с заказчиком) в календарь Lotus Notes и (или) их загрузки из него, необходимо выполнить следующее:

  • Нажмите Сервис->Персональные настройки
  • Откройте пункт Календарь. Укажите Lotus Notes в качестве внешнего календаря.

  • Откройте пункт Интеграция с Lotus Notes. Установочная директория Lotus Notes обычно определяется автоматически, но при необходимости ее можно указать вручную кнопкой Просмотр. Чтобы при работе каждый раз не вводить пароль для Lotus Notes, в этом разделе можно настроить автоматическую авторизацию, указав путь к ID файлу пользователя и пароль. Если путь к ID файлу пользователя автоматически не определился – укажите его вручную кнопкой Просмотр.

  • Откройте пункт Модули к внешним программам. Напротив записи Lotus Notes нажмите кнопку Установить.

После этого запланированное интервью будет автоматически выгружаться в календарь Lotus Notes. При изменении даты/времени этого события в E-Staff в календаре Lotus Notes оно тоже передвинется.

  • Для загрузки запланированных событий из календаря Lotus Notes во внутренний календарь E-Staff используется фоновая задача Загрузка из внешнего календаря. Настройка этой фоновой задачи производится в разделе Администрирование -> Справочники –> Прочие -> Локальные фоновые задачи.

В ручном режиме данная задача запускается из главного меню программы командой Сервис -> Ручной запуск фоновой задачи -> Загрузка из внешнего календаря.

Для настройки загрузки по расписанию (ежедневно или через иной промежуток времени) в полях Час и Минута укажите требуемые значения.

После этого при изменении даты/времени запланированного события в календаре Lotus Notes в календаре E-Staff оно тоже передвинется (после отработки фоновой задачи Загрузка из внешнего календаря в ручном или автоматическом режиме).
