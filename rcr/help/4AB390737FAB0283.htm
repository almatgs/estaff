Интернет-модуль (учетные записи)

Интернет-модуль программы имитирует действия человека при работе с сайтами, экономя время и усилия по заполнению форм на сайтах и по обработке результатов, выдаваемых сайтами. Для настройки интернет-модуля используется раздел Персональные настройки–>Интернет-модуль–>Учетные записи.

  • Откройте карточку работного сайта, который Вас интересует, введите свои регистрационные данные для этого сайта (логин, пароль).

Политика безопасности отдельных работных сайтов требует получения специального доступа для взаимодействующего с ним приложения. Для настройки взаимодействия E-Staff с такими сайтами необходимо выполнить следующее:

  • В карточке настроек работного сайта введите свой логин от учетной записи и нажмите кнопку Запросить доступ для приложения.
  • В открывшемся окне браузера выполните вход, используя свой логин и пароль от учетной записи работного сайта и на странице управления доступом нажмите кнопку Разрешить доступ. После этого скопируйте в буфер обмена сформированный сайтом код. В E-Staff в карточке настроек работного сайта нажмите кнопку Вставить авторизационный код из буфера обмена. Сохраните изменения.


  • Для проверки правильности введенных данных нажмите кнопку Проверить аккаунт. При корректно введенных данных появится сообщение об успешно пройденной проверке. Сохраните изменения.
