Корпоративные дизайны для публикации

Справочник Корпоративные дизайны для публикации содержит список дизайнов Вашей организации, созданных на работных сайтах, для возможности их выбора в разделе Объявление в Интернет карточки вакансии при публикации.

  • для создания карточки нового корпоративного дизайна нажмите кнопку Новый элемент в правом верхнем углу.
  • в поле Наименование введите имя добавляемого дизайна.
  • под заголовком ID на сайтах в поле работного сайта введите идентификатор Вашего корпоративного шаблона, присвоенного на этом сайте.
  • сохраните карточку дизайна кнопкой Сохранить и закрыть.
