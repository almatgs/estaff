Интеграция с календарем Google

Для настройки выгрузки запланированных событий (как правило, Интервью и Интервью с заказчиком) в календарь Google и (или) их загрузки из него, необходимо выполнить следующее:

  • Нажмите Сервис->Персональные настройки
  • Откройте пункт Календарь. Укажите Google в качестве внешнего календаря.
  • Откройте пункт Интеграция с Google. Введите свой логин от учетной записи Google и нажмите кнопку Запросить доступ для приложения.
  • В открывшемся окне браузера выполните вход, используя свой логин и пароль от учетной записи Google и на странице управления календарями нажмите кнопку Принять. После этого скопируйте в буфер обмена сформированный в специальном поле код. Вернувшись в E-Staff в окно настроек интеграции с Google, нажмите кнопку Вставить авторизационный код из буфера обмена. Для проверки правильности введенных данных нажмите кнопку Проверить аккаунт. При корректно введенных данных появится сообщение Проверка выполнена успешно. Сохраните изменения.

После этого запланированное интервью будет автоматически выгружаться в календарь Google. При изменении даты/времени этого события в E-Staff в календаре Google оно тоже передвинется.


  • Для загрузки запланированных событий из календаря Google во внутренний календарь E-Staff используется фоновая задача Загрузка из внешнего календаря. Настройка этой фоновой задачи производится в разделе Администрирование -> Справочники –> Прочие -> Локальные фоновые задачи.

 

В ручном режиме данная задача запускается из главного меню программы командой Сервис -> Ручной запуск фоновой задачи -> Загрузка из внешнего календаря.

Для настройки загрузки по расписанию (ежедневно или через иной промежуток времени) в полях Час и Минута укажите требуемые значения.

 

После этого при изменении даты/времени запланированного события в календаре Google в календаре E-Staff оно тоже передвинется (после отработки фоновой задачи Загрузка из внешнего календаря в ручном или автоматическом режиме).
