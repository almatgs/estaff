Перестройка файлов каталога

В сетевой версии E-Staff

На компьютере, где установлен сервер E-Staff, необходимо выполнить следующее:

  • Запустите E-Staff Server из меню Пуск -> Программы (Start -> Programs) и выключите сервер E-Staff.
  • В меню выберите Сервис->Удалить файлы каталога.
  • Включите сервер и дождитесь перестройки файлов каталога.

 

В однопользовательской версии E-Staff

  • Закройте E-Staff Рекрутер.
  • Удалите папку secondary из папок data_rcr и trash в установочной директории программы (по умолчанию - C:\Program Files\EStaff).
  • Запустите E-Staff Рекрутер и дождитесь перестройки файлов каталога.

 

Дополнительная информация:

  • Доступ к базе данных будет закрыт на время перестройки файлов каталога.
  • Средняя скорость перестройки файлов каталога - 1-3 минуты на 1000 объектов.
