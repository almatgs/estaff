Групповые интервью

 Для создания записи о предстоящем групповом интервью на панели инструментов нажмите кнопку «Новое событие» и выберите пункт Групповое интервью.

В карточке события укажите необходимые данные: дату и время планируемого группового интервью, его продолжительность, регион, макс. кол-во (обязательное поле), типовую должность и нажмите Сохранить и закрыть. Запись о данном событии отобразится во внутреннем календаре E-Staff.

Список добавленных групповых интервью можно посмотреть в разделе События->Групповое интервью.

  1. Занесение новых кандидатов в базу и запись на групповое интервью через Колл-центр

Помимо заведения кандидата одним из стандартных способов, возможно добавление кандидата через специальный интерфейс, предназначенный для сотрудников колл-центра.

Зайдите в основном разделе программы в раздел Прочее->Колл-центр

Для подготовки формы к добавлению новой карточки соискателя нажмите кнопку Новый кандидат

Появится упрощенная форма для ввода данных о кандидате с возможностью завершить добавление одним из типовых действий (например, записью на групповое интервью).

Введите имеющиеся данные о кандидате (ФИО, телефон и т.д.).

Выполните предварительный отбор кандидата, указав:

  • город, в котором он ищет работу (поле Регион поиска работы)
  • типовую должность, на которую он претендует (“продавец”, “кассир” и т.п.) (поле Типовая должность)
  • одну или несколько точек, в каких он хотел бы работать, исходя из территориальной близости (поле Где готов работать)
  • Нажмите кнопку Запись на групповое интервью

Откроется диалоговое окно для выбора группового интервью, в котором будут показаны ближайшие мероприятия, на которых есть свободные места, в выбранном регионе и для выбранной типовой должности.

В списке доступных групповых интервью выберите то, на которое необходимо записать данного кандидата и нажмите ОК.

В базу добавится новая карточка кандидата со статусом «Записан на групповое интервью».

  1. Ввод результатов прохождения группового интервью

Результаты прохождения мероприятия для каждого из кандидатов отмечаются путем выбора одной из нескольких галочек (пройдено, не пройдено, самоотказ, не пришел, и т.п.) либо в личном деле кандидата, либо в списке участников в карточке группового мероприятия, открыв ее в разделе События-Групповое интервью.
