Шаблоны SMS

Программа E-Staff Рекрутер позволяет отправлять короткие сообщения (SMS) отобранным кандидатам и заказчикам на мобильный телефон по заранее подготовленным шаблонам. Для выбора требуемого шаблона для отправки SMS Вы можете использовать кнопку с изображением моб.телефона в карточке кандидата или заказчика.

Настройка функции отправки SMS осуществляется в разделе Администрирование–>Справочники–>Интеграция–>SMS-шлюзы. Для правильной настройки может потребоваться помощь Вашего системного администратора. В отдельных случаях настройку сможет произвести только системный администратор.

Для добавления шаблонов SMS используется раздел Администрирование->Справочники->Шаблоны SMS.

  • Для добавления нового шаблона нажмите кнопку Новый элемент.
  • В открывшемся окне введите Наименование шаблона (Вы будете видеть его в списке при выборе шаблона SMS для отправки), укажите Тип получателя (шаблон для контактного лица, кандидата, пользователя или фиксированного номера), и наберите текст сообщения. Если выбран тип получателя - Пользователь, то необходимо проверить, что в настройках пользователей (раздел Администрирование->Доступ->Пользователи) указан соответствующий сотрудник и в карточке этих сотрудников вписан мобильный телефон.

В тексте сообщения Вы можете использовать стандартные вычисляемые элементы (ФИО, вакансия, дата запланированного события), на место которых программа будет подставлять фактические значения.

Примечание: для корректного формирования сообщений с вычисляемыми элементами необходимо, чтобы на момент создания сообщения в карточке присутствовали данные, которые подставляет тот или иной вычисляемый элемент.

Если создаваемый шаблон не предназначен для отправки SMS вручную из карточки кандидата или заказчика, установите флажок Скрыть шаблон из списка доступных для выбора пользователем.

Если требуется чтобы факт отправки сообщения по данному шаблону фиксировался, установите флажок Регистрировать факт отправки в виде события и выбрать необходимое событие.

 

Флажок Новая группа в меню позволяет при открытии списка шаблонов в карточке отделить этот шаблон вместе с теми, которые ниже его в общем списке шаблонов, от остальных.

Указав группу в поле Использовать только внутри группы пользователей, можно ограничить доступность текущего шаблона для пользователей пределами указанной группы.

Поскольку в многопользовательской версии программы все пользователи используют общие шаблоны, каждому пользователю необходимо заполнить шаблон своей подписи в разделе Сервис->Персональные настройки (пункт Подпись).
