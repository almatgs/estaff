Внешние учетные записи

В сетевой версии программы администратор приложения имеет возможность централизованного добавления регистрационных данных пользователей от работных сайтов. 

  • Откройте раздел Администрирование->Сервис->Внешние учетные записи.
  • Для добавления новой учетной записи работного сайта нажмите кнопку Новый элемент.
  • В поле Тип Выберите Работный сайт.
  • В поле Работный сайт выберите сайт, для взаимодействия с которым производится настройка.
  • В поля Логин, Пароль введите данные от учетной записи пользователя, под которыми он зарегистрирован на данном работном сайте.
  • В поле Пользователь выберите пользователя программы – владельца данной учетной записи.

Для проверки правильности введенных данных нажмите кнопку Проверить аккаунт. При корректно введенных данных появится сообщение об успешно пройденной проверке. Сохраните изменения.
