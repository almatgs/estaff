Общие принципы работы с разделами программы

Все разделы программы, перечисленные в левой части главного окна, построены по единому принципу.

В правой части экрана отображается список объектов, соответствующих выбранному разделу (подразделения, вакансии, кандидаты и т.д.)

·         Для изменения порядка сортировки объектов щелкните мышкой по заголовку колонки, по которой должна производиться сортировка.

·         Для изменения порядка сортировки на обратный повторно щелкните мышкой по заголовку колонки.

·         Для открытия карточки объекта дважды щелкните мышкой на соответствующую строку в списке.

·         Для удаления карточки объекта выделите соответствующую строку в списке, затем нажмите клавишу Delete.

 

Также для открытия или удаления объект можно использовать контекстное меню, выпадающее при нажатии правой кнопки мыши на строку в списке.

 

Над списком отображается набор полей, по которым можно осуществлять фильтрацию объектов в списке. Первым всегда идет полей Поиск по тексту, позволяющее указать любое ключевое слово (или фрагмент) слова, которое должно содержаться внутри карточки. Для получения более подробной информации см. раздел Поиск по ключевым словам.

·         Для фильтрации записей задайте нужны набор значений фильтров в верхней части экрана над списком.

При этом, в случае выбора фильтра с выпадающим списком значений, фильтр применяется сразу после выбора значения. В случае фильтра с вводом текста, для применении фильтра необходимо нажать кнопку Найти, или, находясь в текстовом поле, нажать клавишу Enter.

 ·         Для сброса всех используемых фильтров необходимо правой кнопкой мыши нажать в свободную область вокруг фильтров и выбрать Очистить форму.
