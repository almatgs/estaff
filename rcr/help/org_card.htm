Карточка организации

Карточка организации в программе E-Staff Рекрутер имеет следующую структуру:

В левой части карточки (под содержанием раздела) расположен список ролей, которые могут быть присвоены организации в зависимости от функций, выполняемых ею по отношению к Вашей компании. Организации в роли Конкурент впоследствии могут быть использованы для указания дополнительного источника поступления кандидатов (поле Компания-донор раздела Доп. сведения карточки кандидата).

Флажок Источник поступления кандидатов (только для режима "Кадровая служба компании") позволяет автоматически добавить данную организацию в качестве нового источника в справочник Кандидаты->Источники кандидатов.

 

Раздел Общие сведения содержит основную информацию об организации.

Поле Номер заполняется программой автоматически, тем не менее, Вы можете исправлять его вручную. Если Вас не устраивает предлагаемый программой способ нумерации, Вы можете изменить его в разделе Администрирование->Объекты.

Поле Условное наименование содержит наименование организации, предназначенное для внутреннего использования в программе, например, Главстройпроект.

Поле Полное (официальное) наименование содержит наименование организации для использования в официальных документах, например, ОАО "Главстройпроект".

Поле Альт. наименование содержит альтернативное наименование организации, по которому ее можно будет найти (как правило, это сокращенное или иностранное наименование), например, ГСП.

Поле Адрес содержит фактический адрес организации.

Поле Страна содержит наименование государства, в котором находится организация (значение из справочника).

Поле Регион содержит регион, в котором находится организация (значение из справочника).

Поля Телефон, Факс, E-Mail, Web-сайт содержат общую контактную информацию по организации, не привязанную к какому-либо определенному контактному лицу.

Поле Контактные лица содержит список контактных лиц по данной организации.

Поля Сфера деятельности, Размер, Категория классифицирует организацию по соответствующим настраиваемым справочникам.

Поле Дата последн. контакта содержит дату последнего контакта с данной организацией. Если поле Дата последн. контакта у какого-либо из контактных лиц становится больше, чем аналогичное поле по организации, поле Дата последн. контакта по организации обновляется до соответствующего значения.

Поля Всего вакансий, Активных вакансий показывают количество вакансий (общее и в работе соответственно) по данной организации.

Поле Доп. сведения может содержать любую произвольную дополнительную информацию.

 

Раздел Реквизиты содержит реквизиты организации как юридического лица.

Раздел Активные вакансии содержит список открытых на данный момент вакансий по данному клиенту (только для режима "Рекрутинговое агентство").

Раздел Все вакансии содержит список всех вакансий по данному клиенту (только для режима "Рекрутинговое агентство").

Раздел События по кандидатам содержит список всех событий по кандидатам, рассматриваемых на вакансии по данному клиенту (только для режима "Рекрутинговое агентство").

Раздел Приложения может содержать произвольные приложения к карточке организации.
