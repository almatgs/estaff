Направления подбора

При совместном использовании модулей массового и классического подбора возможно разделить работу с кандидатами и вакансиями по направлениям (например, “Специалисты”, “Высший менеджмент”, “Поток”, “Стажеры” и т.п.). При этом для каждого из направлений можно задать свой список статусов кандидатов. Для этого в настройках требуемого события (раздел Доп. сведения) в поле Направления подбора необходимо выбрать те направления, при работе по которым будет использоваться данное событие.

Определенную группу рекрутеров можно привязать к определенным направлениям подбора. Для этого в настройках группы в поле Направления подбора необходимо указать направление, которым будет заниматься данная группа пользователей.
