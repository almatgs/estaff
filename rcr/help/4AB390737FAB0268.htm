Поиск кандидатов

В разделе Кандидаты->Поиск кандидатов можно осуществлять поиск по всей базе резюме по самым различным критериям, включая поиск по ключевым словам и событиям, занесенным в личное дело кандидатов за указанный период.

Для осуществления поиска  задайте необходимые критерии поиска, используя имеющийся набор фильтров и нажмите кнопку Найти. По окончании поиска в нижней части окна будет выдан список найденных кандидатов.

Для поиска кандидатов по фамилии, имени или отчеству рекомендуется использовать специальное поле ФИО.

С помощью поля Поиск по тексту можно производить поиск кандидатов по одному либо нескольким ключевым словам (в т.ч. по номеру телефона или его началу с кодом региона либо без него), которые должны присутствовать внутри карточки (см. Поиск по ключевым словам).

Дополнительно возможен поиск по наличию приложений определенного типа в карточке кандидата. Для этого необходимо будет включить динамический фильтр Наличие приложений с помощью команды Вид->Фильтры.

При использовании почтовой программы Microsoft Outlook или Mozilla Thunderbird можно непосредственно из нее проверять наличие кандидата в базе по электронному адресу отправителя (e-mail) с помощью специальной кнопки. Выбрав письмо, нажмите на кнопку Найти в E-Staff – будет произведен поиск и при наличии в базе кандидата,  имеющего такой же e-mail, программа автоматически откроет его карточку. Указанная кнопка входит в состав надстройки для почтовых программ Microsoft Outlook и Mozilla Thunderbird, устанавливаемой в разделе Персональные настройки->Модули к внешним программам.
