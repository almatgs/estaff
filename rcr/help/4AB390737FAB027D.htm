Группы

Заводить группу в сетевой версии программы может либо администратор системы, либо другой пользователь с правами "Администратор приложения".

  • Для заведения новой группы перейдите в раздел Администрирование->Доступ->Группы и нажмите кнопку Новая группа.

Откроется карточка новой группы:

  • Введите Наименование группы пользователей.
  • Поле Головная группа содержит ссылку на головную (родительскую) группу. Поле позволяет сформировать иерархию групп. Для групп верхнего уровня это поле не заполняется.

В поле Дочерние группы отображается список групп, для которого данная группа является головной.

Для автоматического распределения заявок, поданных через портал WebTutor или внешнюю систему кадрового учета, по определенным рекрутерам и группам (в зависимости от позиции вакансии в организационной структуре) в поле Подразделения, за которые отвечает данная группа нужно будет указать требуемые подразделения, и ответственного рекрутера (поле Рекрутер, обрабатывающий заявки на вакансии, поданные из внешней системы). В этом случае заявки, поступающие из указанных (и дочерних) подразделений, будут закрепляться за соответствующей ответственной группой и рекрутером. 

Поле Включает в себя права групп позволяет при необходимости расширить область действия прав пользователей основной группы, распространив их и на другие группы, т.е. предоставить доступ к объектам и других групп, добавив эти группы кнопкой Выбрать группы.

В поле Комментарий при необходимости можно ввести произвольную дополнительную информацию по группе.
