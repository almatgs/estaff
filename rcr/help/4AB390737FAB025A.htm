Навыки

Справочник Навыки содержит список навыков, используемых в системе для быстрого указания их в карточке кандидата.

По умолчанию справочник заполнен списком иностранных языков.

Создание простого навыка

  • нажмите кнопку Новый элемент.
  • введите наименование, установите флажок Уровни и с помощью кнопки «+» (Добавить уровень) добавьте необходимое количество уровней, характеризующих создаваемый навык.

Создание группы навыков

  • нажмите кнопку Новый элемент для создания родительского элемента группы навыков.
  • введите наименование, установите флажки Уровни и Группа навыков (последний используется, чтобы программа запрещала указывать в качестве навыка головной группирующий элемент - это необходимо для тех случаев, когда уровень владения должен указываться для конкретного навыка) и с помощью кнопки «+» (Добавить уровень) добавьте необходимое количество уровней, характеризующих создаваемую группу навыков.
  • нажмите кнопку Новый элемент для создания подчиненного элемента группы навыков.
  • введите наименование, укажите соответствующий родительский элемент, установите флажки Уровни и Использовать уровни головной группы (последний используется, чтобы к данному элементу применились те же уровни владения навыком, что указаны в родительском элементе).

Как пример, можно рассмотреть настройки уже включенной в программу группы навыков Иностранный язык. В головном элементе Иностранный язык установлен флажок Группа навыков, чтобы программа запрещала указывать в качестве навыка данный элемент (т.к. уровень владения должен указываться для конкретного языка) и введены уровни, которые будут использовать все подчиненные объекты.

В созданных подчиненных элементах (английский, немецкий и т.д.) указан соответствующий родительский элемент (Иностранный язык) и установлен флажок Уровни, так как для этих элементов будут использоваться характеризующие их уровни. Иностранных языков много, но перечень уровней владения для каждого языка одинаков, поэтому в этих подчиненных элементах установлен флажок Использовать уровни головной группы.
