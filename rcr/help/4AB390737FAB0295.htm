Региональные настройки

Раздел Региональные настройки позволяет задать требуемые параметры локализации:

  • Страна.
  • Вид основной валюты (для хранения уровня заработной платы).
  • Режим мультивалютности. В области Разрешить выбор из нескольких валют для полей включением опций можно активировать возможность выбора вида валюты при указании уровня зарплаты в карточке кандидата, вакансии или штатной должности. Наполнение справочника валют можно изменить в разделе Администрирование->Справочники->Прочие-Валюты.
  • Основной регион. По умолчанию не указан. Используется для автоматического указания региона поиска работы при ручном заведении карточек кандидатов.
  • Собственные почтовые домены. В этом поле можно ввести собственные почтовые домены вашей компании. Если резюме будет помещаться из почтовой программы от адресатов, имеющих указанные домены (например, из сообщения с резюме соискателя, которое переслал вам кто-то из коллег), то поле E-Mail в карточке кандидата будет заполняться только его электронным адресом, а собственные адреса компании будут игнорироваться.
  • Режимы проверки формата и форматирования вводимых в карточки телефонных номеров.
