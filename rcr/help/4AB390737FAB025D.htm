Специальности

По умолчанию программа использует двухуровневый справочник специальностей, поставляемый вместе с программой.

Вы можете вручную отредактировать стандартный справочник специальностей в разделе Администрирование–>Справочники–>Общие–>Специальности.

Если справочник специальностей представлен в формате программы Microsoft Excel (XLS), то его возможно сформировать автоматически, загрузив данные из такого xls-файла в E-Staff.

Внимание! При выполнении описанного импорта стандартные элементы справочника специальностей будут удалены.

  • Для загрузки справочника специальностей из xls-файла нужно зайти в раздел Администрирование–>Справочники–>Общие–>Специальности и нажать кнопку Импорт из MS Excel.
  • В открывшемся окошке указать xls-файл и нажать кнопку Открыть.
  • Запустится импорт и после его завершения в разделе Специальности появится подгруженный справочник специальностей.

Формат файла XLS для импорта специальностей

Первый столбец файла является корневым, т.е. может содержать специальность и (или) наименование категории (для построения иерархического справочника специальностей). Каждый последующий столбец является подчиненным, т.е. может содержать построчный набор специальностей (завершение ветви иерархии) и (или) наименование подкатегории (углубление иерархии).
