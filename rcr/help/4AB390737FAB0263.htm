Карточка договора

Карточка договора в программе E-Staff Рекрутер имеет следующую структуру:

Поле Дата содержит дату заключения договора с контрагентом.

Поле Номер содержит номер договора с контрагентом.

Поле Контрагент содержит ссылку на контрагента, с которым заключен договор.

Поле Срок действия содержит общий срок действия договора.

Поле Сумма содержит общую сумму договора.

Поле Валюта содержит ссылку на вид денежной валюты, используемой в договоре (выбирается из справочника валют).

Поле Условия договора может содержать произвольную дополнительную информацию об условиях заключенного договора.

Поля Дата, Номер, Контрагент являются обязательными для заполнения.
