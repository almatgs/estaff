Employee Card

Each employee card in E-Staff has the following structure:

General section:

#: Internal identification number. By default this field is filled in automatically, but you can change it any time. How numbers are assigned can be set up in Administration->Objects section.

Last name: Last name of the employee.

First name: First name of the employee.

Gender: Gender of the employee. Filled in automatically if the first name is found in the program database.

Birth Date: Date of birth of the employee.

Work Phone, Mobile Phone: Contact phones.

E-Mail: Primary (usually corporate) e-mail. Primary E-mail is used when sending notifications.

E-Mail 2: Alternate (usually personal) e-mail.

Division: Reference to division the employee works in.

Position: Name of position. This field may be ether a free text or a reference to a list of positions, depending on the settings in Administration->Global Settings->Organization Structure.

Head of Division: Flag specifying that the employee is a head of the division.  This flag may be set automatically if Require binding each employee to a position option is checked in Administration->Global Settings->Organization Structure.

Comment: Any additional information.

 Add. Info section:

External ID: If organization structure is loaded automatically from a corporate portal or employees tracking system, this field contains an identifier of this employee in the external system.

Previous Jobs section: Contains the list of the employee’s previous jobs .
