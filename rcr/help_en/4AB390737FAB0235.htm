Managing organizational structure

When used by the HR department, E-Staff provides a way to manage the organization and personnel structure, including divisions, positions, and employees. The data may be stored with various levels of detail, depending on the users' needs.

It is possible to load the organization and personnel structure automatically from the personnel management system (if an integration has been set up), or to complete if manually using the File menu. If you enter organization and personnel structure manually, you may complete it gradually, along with appropriate vacancies.

Org. structure section contains a hierarchical list of company divisions.

Employees section contains a list of all employees in a company.

Positions section contains a list of all staff positions in a company. The presence of this section is optional, depending on E-Staff usage policy, specified with Use positions checkbox in Administration->Global section->Organization structure. When using positions, each employee may (or must) be bound to a position, which is freed when an employee is dismissed. Each vacancy may (or must) be bound to a position as well. When not using positions, you are free to assign any name to a vacancy or any position title to an employee.

When using positions, to prohibit assigning arbitrary names to vacancies or employee position titles

  • Check Require binding each employee to a position checkbox in Administration->Global section->Organization structure section and Require binding each vacancy to a position checkbox in Administration->Global section->Vacancies section, respectively.

You may also enable using standard positions, so that each position may (or must) be linked to a standard position.

Enable standard positions by

  • Checking Use standard positions checkbox in Administration->Global section->Organization structure.

Standard positions are rarely used, as it is typical to store a database of unified position descriptions.
