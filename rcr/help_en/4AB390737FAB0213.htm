Adding a candidate manually

Manually adding a candidate is uncommon, but can be done.

  • Click the New Candidate toolbar button.

An empty candidate card opens.

  • Use the fields shown above to enter Last Name, First Name, Address, etc.

Note: Ensure that you fill in the Last Name and First Name fields, as they are displayed in the standard candidate lists.

For more information about filling in  fields, see the Candidate card section.

E-Staff prompts you to save the information:

Click Yes.

You can also save by selecting Save from the File menu or pressing Ctrl+S.
