Importing Candidates using Microsoft Outlook toolbar button

E-Staff allows you to add the Import CV to E-Staff button to the Microsoft Outlook toolbar. With this button you can import selected messages containing résumés to E-Staff with a single click. Both résumés from a message text and from attached files are processed.

 E-Staff, when first launched, prompts you to add the button automatically. Additionally, you may add the button using Personal Settings->External application plug-ins section.

 

To import a résumé from a mail message to E-Staff

  • Select the messages containing résumés.
  • Click Import CV to E-Staff toolbar button.

A new candidate card opens. The successfully parsed fields are already filled in. The lower part of the screen displays the source résumé text.

Before importing, E-Staff automatically checks for duplicates. If possible duplicates are found, E-Staff displays a warning message.
