Using object cards

All object card have a common structure.

Candidate's card usually contains resume as a single attachment. Additionally, you can add any number of attachments (additional information, job description, location map, etc.) to any object's card.

To add an attachment:

·         Open candidate's, vacancy's or company's card, then click Attach toolbar button and select attachment type.

En empty attachment opens. You can edit an attachment manually, copy the text from other editor, or import from Microsoft Word using Import button.
