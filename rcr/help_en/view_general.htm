General Guidelines

The left tree of the main program window contains all the program sections. Right part of the window is usually a list of objects of appropriate type: vacancies, candidates, events, etc.

  • To change the sorting field click in the title of a column to sort for.
  • To change the sorting direction (ascending or descending) click again in the title of a column.
  • To open an object card displayed in the list, double-click it.
  • To delete an object displayed in the list, select it and press Delete.

You can use right-click context menu as well.

 

A set of filters is located above the object's list. The first filter is always Search Text, which provides searching for any keyword inside an object card. See Fulltext Search section for details.

A filter with a drop-down list applies automatically. A text filter applies when you click the Find button or press Enter.

  • To clear all the filters, right click on the free area between the filters and select Clear Form.
