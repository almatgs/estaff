Importing Candidates from mailboxes

E-Staff provides tools to import emailed résumés or CVs, whether from email message text or a Word (OpenOffice) attachment.

Because of the variance in email programs,  some additional setup may be needed to import résumés from certain programs. The setup itself is performed in the Import From Mailboxes section of the user settings dialog (Tools menu, User Settings item). You may need assistance from your system administrator to set up this feature.  

 E-Staff can be set up to handle email messages in different modes depending on whether the mailbox to be processed is a dedicated résumé mailbox. E-Staff is able, with a high success rate, to distinguish between an email that is a résumé and one that is not.

How the standard E-Staff settings handle mailbox import:

  • E-Staff opens a mailbox and scans all unread letters.
  • If E-Staff finds a letter with a résumé, this letter, both résumé and text, is entered in a preliminary import list, while the letter itself is marked as “read” in the mailbox.
  • If E-Staff recognizes that a letter does not contain a résumé, it leaves it alone.

 

To run Résumé Import from a mailbox

  • Click the Resume Processing/From Mailboxes item in the left side of the E-Staff main window.
  • Click Start Import to begin importing.

E-Staff begins to scan mailboxes and process the résumés it finds. As the scan progresses, the imported résumés appear in the list in the upper part of the window.

Résumés are parsed; for a single résumé, this takes 2-5 seconds depending on your computer. While the import runs, you may switch to other sections and continue to work with E-Staff.

 

To view an imported résumé

 

  • Click its name in the list

    The text of the letter and all its attachments (on separate tabs) are displayed in the lower part of the window.

    Note: All imported résumés are put in a TEMPORARY list and are not entered in the main candidate database. You may either add single or several résumés to the candidate database or delete them. Select a group of résumés using either the Shift or Ctrl keys.

    To add selected résumé to the main database

     

  • Click Save in database

     

    To delete one or several resumes

     

  • Click the Delete button located above the list or press Del.
