Hardware and Software Requirements

Single-user version, network version client:

  • Microsoft Windows 2000, XP, 2003 Server, Vista, 2008 Server, Windows 7
  • 512 Mb RAM
  • Microsoft Office 2000 or later (for importing résumés from Microsoft Word or integration with Microsoft Outlook calendar), or OpenOffice.org 2.3 or later (for importing résumés from OpenOffice.org)

 Network version server:

  • Microsoft Windows 2000, XP, 2003 Server, Vista, 2008 Server, Windows 7
  • 1 Gb RAM
