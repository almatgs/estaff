Importing Candidates using Internet Explorer toolbar button

E-Staff allows you to add Import to E-Staff button to Internet Explorer's toolbar. With this button you can import selected parts of a Web page as a résumé with a single click.

E-Staff, when first launched, prompts you to add the button. You may also add the button using Personal Settings->External application plug-ins section.

 To import a résumé from a Webpage to E-Staff

  • Select the part of the Web-page containing résumé.
  • Click Import to E-Staff toolbar button.

A new candidate card opens. The successfully parsed fields are already filled in. The lower part of the screen displays the source résumé text.

Before importing, E-Staff automatically checks for duplicates. If possible duplicates are found, E-Staff displays a warning message.
