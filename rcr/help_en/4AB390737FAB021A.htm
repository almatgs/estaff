Selecting candidates for vacancies

When a vacancy registered in your database is active, you may select candidates for this vacancy.

To select a candidate for a vacancy

 

  • Open the candidate card and click the Select for Vacancy button in the upper part of the card.

    •  

    If there are active vacancies in your database, a dialog window displaying the active vacancies opens.

    To select this candidate for one of the displayed vacancies

     

  • Select the vacancy you want
  • Click OK.

    • Note: The same candidate can be selected for several vacancies, with separate selection states for each vacancy.
