Setting up the Personal version

Setting up the Personal version

  • Run the EStaff_EN.exe setup file.

A setup dialog window opens.

  • Click Next.

A prompt for the serial number opens.

  • Enter the serial number of your product copy (4 symbols in each field)
  • Click Next.

A prompt for the program setup folder opens that contains the default setup folder. Typically, the default setup folder does not need to be changed.

  • Specify the folder where you want to install E-Staff
  • Click Next.

The files copy from the installation disk to your hard disk (this takes about a minute).

When installation is complete

  • Click “Finish”

A shortcut to E-Staff is added to your Desktop and to the Start/Programs menu.

  • Launch E-Staff trough the Desktop shortcut or using Start/Programs menu.

A dialog window with an activation prompt opens.

If your computer has an Internet connection

  • Select Automatic activation
  • Click OK.

E-Staff will connect to the developer's site and obtain an activation code for your computer.

If Internet connection is not available

  • Select Manual activation.

A manual activation window opens. Call your product seller and give them the installation code.

  • Enter the activation code given by the seller.

If you start E-Staff for the first time, the Initial Setup Wizard opens. It is recommended that you set up the following parameters:

  • Usage type (HR department or recruiting agency)
  • Vacancy management policy
  • Default currency

To change these parameters later, use Administration->Global settings sections.
