Setting up the Network version

  • Run the EStaff_Server_EN.exe setup file.

A setup dialog window opens.

  • Click Next.

A prompt for the serial number opens.

  • Enter the serial number of your product copy (4 symbols in each field)
  • Click Next.

A prompt for the program setup folder opens that contains the default setup folder, which typically does not need to be changed.

  • Click Next.

Files now copy from the installation disk to your hard disk (this takes about a minute).

When installation is complete

  • Click Finish.

A shortcut to E-Staff Server is added to your Desktop and to the Start/Programs menu.

  • If you have used the single-user version before, copy "data_rcr" folder from C:\Program Files\EStaff to C:\Program Files\EStaff_Server folder.
  • Launch E-Staff Server through the Desktop shortcut or using Start/Programs menu.
  • Enter administrator login and password (you may choose any).
  • Click Start.

A dialog window with an activation prompt opens.

If your computer has an Internet connection

  • Select Automatic activation
  • Click OK.

E-Staff will connect to the developer's site and obtain an activation code for your computer.

If an Internet connection is not available

  • Select Manual activation.

A manual activation window opens. Call your product seller and provide the installation code. Enter the activation code given by the seller.


 

Setting up the Client software

  • Run the EStaff_Client_EN.exe setup file.

A setup dialog window opens.

  • Click Next.

A prompt for the program setup folder opens that contains the default setup folder. Typically, this does not need to be changed.

  • Click Next.

The files now copy from the installation disk to your hard disk (this takes about a minute).

When installation is complete

  • Click Finish

A shortcut to E-Staff is added to your Desktop and to the Start/Programs menu.

 

Launching the Client software on the First Computer

  • Launch E-Staff through the Desktop shortcut or using Start/Programs menu.
  • Enter server computer name or IP address. The user authorization dialog should appear.
  • Login as administrator.

When starting  E-Staff for the first time, the Initial Setup Wizard opens. It is recommended that you set up the following parameters:

  • Usage type (HR department or recruiting agency)
  • Vacancy management policy
  • Default currency

You can change these parameters later using Administration->Global settings sections.

  • Add other users in Administration->Users section.
  • Quit the program, open it again, then login under your regular user name.
