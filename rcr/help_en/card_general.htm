Using Object Cards

All object cards have a common structure.

A Candidate's card usually contains the resume as a single attachment. Additionally, you can add any number of attachments (additional information, job description, location map, etc.) to any object's card.

To add an attachment

  • Open the candidate's, vacancy's or company's card
  • Click the Attach toolbar button and select attachment type.

An empty attachment opens. You can edit an attachment manually, copy the text from other editor, or import from Microsoft Word or OpenOffice using Import button.
