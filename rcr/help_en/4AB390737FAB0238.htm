Single-User and Multi-User Versions

E-Staff is provided in both single- and multi-user versions.

The single-user version, intended for use by only one person, is installed on a single computer, which stores the database.

The multi-user version, intended for use by multiple persons simultaneously, grants each user his/her own access level for different parts of a database or different program sections. The multi-user version has a server component that should be installed on a server computer, which will store the data. The server computer may be a dedicated server machine or, for a small number of users, one of the user’s workstations.
