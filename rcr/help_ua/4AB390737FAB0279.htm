Шаблони SMS

Програма E-Staff дозволяє відправляти короткі повідомлення (SMS) відібраним кандидатам та замовникам на мобільний телефон за заздалегідь підготовленим шаблонам. Для вибору потрібного шаблону для відправлення SMS Ви маєте можливість використати кнопку із зображенням мобільного телефону в картці кандидата або замовника.

Налаштування функції відправки SMS здійснюється в розділі Адміністрування–>Довідники–>Інші–>SMS-шлюзи. Для коректного налаштування можливо знадобиться допомога Вашого системного адміністратора. В окремих випадках налаштування може провести тільки системний адміністратор.

Для додавання шаблонів SMS використовується розділ Адміністрування->Довідники->Шаблони SMS.

  • Для додавання нового шаблону натисніть на кнопку Новий елемент.
  • У новому вікні введіть Назву шаблону (Ви побачите його у списку під час вибору шаблону SMS для відправки), вкажіть Тип отримувача (шаблон для контактної особи, кандидата, користувача або фіксованого номера), та наберіть текст повідомлення. Якщо обрано тип отримувача – Користувач, то необхідно перевірити, що в налаштуваннях користувачів (розділ Адміністрування->Доступ->Користувачі) вказаний відповідний співробітник та в картці цих співробітників вписаний мобільний телефон.

В тексті повідомлення Ви маєте можливість використовувати стандартні обчислювальні елементи (ПІБ, вакансія, дата запланованої події), на місце яких програма буде підставляти фактичні значення.

Примітка: для коректного формування повідомлень з обчислювальними елементами необхідно, щоб на момент створення повідомлення в картці були наявними дані, які підставляє той чи інший обчислювальний елемент.

Якщо новий шаблон не призначений для відправлення SMS вручну з картки кандидата або замовника, встановіть перемикач Сховати шаблон із списку доступних для вибору користувачем.

Якщо необхідно, щоб факт відправлення повідомлення за даним шаблоном фіксувався, встановіть перемикач Реєструвати факт відправлення у вигляді події та оберіть необхідну подію.

Перемикач Нова група в меню дозволяє під час відкриття списку шаблонів в картці відокремити даний шаблон разом з тими, які нижче нього в списку шаблонів, від решти.

Вказавши групу в полі Використовувати тільки всередині групи користувачів, можливо обмежити доступність поточного шаблону для користувачів межами вказаної групи.

Оскільки у мережевій версії програми всі користувачі використовують загальні шаблони, кожному користувачу необхідно заповнити шаблон свого підпису в розділі Сервіс->Персональні налаштування (пункт Підпис).
