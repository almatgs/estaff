Шаблони вакансії

Довідник Шаблони вакансій містить перелік створених користувачем шаблонів вакансій для можливості швидкого заповнення розділу Об’ява в мережі Інтернет картки вакансії за шаблоном, який було підготовлено заздалегідь. Для вибору шаблону використовується поле Шаблон для публікації у розділі Додаткові дані  картки вакансії.

Створення власних шаблонів вакансій відбувається у розділі Адміністрування–>Довідники–>Вакансії–>Шаблони вакансій  за допомогою Новий елемент у правому верхньому куті.

Картка шаблона вакансії має наступну структуру:

Розділ Загальні дані:

Назва -  назва вакансії.

Спеціальність – класифікатор об’єктів по спеціальностям (значення з довідника).

Регіон -  регіон майбутнього місця роботи (значення з довідника). Використовується в тому випадку, якщо компанія веде свою діяльність у декількох регіонах.

Розділ Об’ява в мережі Інтернет:

Вкладки Загальне, Опис, Відокремлений опис – використовуються для заповнення інформацією, необхідною для розміщення. Поля вкладки Відокремлений опис використовуються для заповнення під час розміщення вакансії на сайті superjob.ru та стають доступними за наявністю перемикача superjob.ru.
