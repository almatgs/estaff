SMS-шлюзи

Для коректного налаштування даного розділу можливо знадобиться допомога Вашого системного адміністратора. В окремих випадках налаштування може зробити тільки адміністратор.

Додавання SMS-шлюзів відбувається у розділі Адміністрування–>Довідники–>Інші–>SMS-шлюзи за допомогою кнопки Новий елемент в правому верхньому куті.

Поле Назва використовується тільки для візуальної відмінності між декількома агрегаторами та може містити довільний текст.

Поле Скрипт для відправлення SMS призначений для введення програмного коду сценарію, який розроблюється індивідуально для певного SMS-агрегатора, з яким Ви працюєте.

Для перевірки коректності введених даних в полі Тестовий номер телефону вкажіть номер мобільного телефону та натисніть кнопку Відправити тестове повідомлення. При вірних введених даних на вказаний номер прийде SMS-повідомлення з текстом Test.
