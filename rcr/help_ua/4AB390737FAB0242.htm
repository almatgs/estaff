Регіони

Довідник Регіони містить перелік регіонів (обласні центри, області, районні центри декількох держав), які використовуються в системі.

Даний довідник має механізм завантаження, який налаштовується. Для заповнення довідника необхідними для роботи регіонами виконайте наступні дії:

  • Натисніть кнопку Завантажити. Відкриється вікно завантаження довідника регіонів.
  • Оберіть державу та вкажіть тип регіонів, які необхідно завантажити, перемиканням режиму Глибина завантаження. В нижній частині вікна з’явиться відповідний перелік регіонів (попередньо виділених).
  • Для заповнення довідника повним списком регіонів згідно вказаним налаштуванням натисніть на кнопку ОК.
  • Для заповнення довідника тільки окремими регіонами оберіть їх в списку та натисніть на кнопку ОК.
