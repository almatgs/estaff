Картка витрат

Картка витрат в програмі E-Staff має наступну структуру:

Поле Контрагент містить посилання на контрагента, з яким було укладено договір.

Поле Стаття витрат містить посилання на статтю витрат (обирається з довідника статей витрат).

Довідник статей витрат заповнюється користувачем у розділі Адміністрування->Довідники->Загальні->Статті Витрат.

Поле Договір містить посилання на номер договору, який було укладено з контрагентом.

Поля Період надання послуги (з, по) містить дати початку та закінчення надання послуг, які описані в договорі.

Поля Структурний підрозділ, Вакансія, Кандидат містять відповідні посилання для визначення об’єкту, з яким пов’язана дана витрата.

Поле Сума платежу містить загальну суму платежу за даною витратою.

Поле Валюта містить посилання на вид грошової валюти, яка використовується для оплати (обирається із довідника валют).

Поле Статус платежу містить посилання на статус платежу, який було здійснено (обирається з довідника статусів платежів).

Поле Дата оплати містить дату платежу, який було здійснено за певною витратою.

Поле Номер платежу містить номер платіжного документу.

Поле Дод. дані містить довільну додаткову інформацію по даній витраті.
