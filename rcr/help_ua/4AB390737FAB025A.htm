Навички

Довідник Навички містить перелік навичок, які використовуються в системи для швидкого вибору їх в картці кандидата.

За замовчуванням довідник містить перелік іноземних мов.

 

 

Створення простої навички

  • Натисніть на кнопку Новий елемент.

  • Введіть назву, встановіть перемикач Рівні та за допомогою кнопки «+» (Додати рівень) додайте необхідну кількість рівнів, які характеризують нову навичку.

    Створення групи навичок

  • Натисніть на кнопку Новий елемент для створення батьківського елемента групи навичок.

  • Введіть назву, встановіть перемикач Рівні та Група навичок (останній використовується для заборони програмою вказування у якості навички головний елемент, який групує – це необхідно для тих випадків, коли рівень володіння повинен вказуватися для визначеної навички) та за допомогою кнопки «+» (Додати рівень) додайте необхідну кількість рівнів, які характеризують нову групу навичок.

  • Натисніть на кнопку Новий елемент для створення підлеглого елементу групи навичок.

  • Внесіть назву, вкажіть відповідний батьківський елемент, встановіть перемикач Рівні та Використовувати рівні головної групи (останній використовується для того, щоб для даного елемента застосовувались ті ж самі рівні володіння навичкою, які вказані у батьківському елементі).

    Як приклад, можна розглянути налаштування вже включеної до програми групи навичок Іноземна мова. В головному елементі Іноземна мова встановлено перемикач Група навичок, щоб програма забороняла вказувати у якості навички даний елемент (так як рівень володіння повинен вказуватися для конкретної мови) та внесені рівні, які будуть використовуватися для всіх підлеглих об’єктів.

    В створених підлеглих елементах (англійська, німецька та інші) вказаний відповідний батьківський елемент (Іноземна мова) та встановлено перемикач Рівні, так як для цих елементів будуть використовуватися рівні, які їх характеризують. Іноземних мов багати, але перелік рівнів володіння для кожної мови однаковий, тому в цих підлеглих елементах встановлений перемикач Використовувати рівні головної групи.
