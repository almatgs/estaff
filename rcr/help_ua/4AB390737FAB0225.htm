Додаткові поля

Ви маєте можливість розширювати стандартні картки кандидата, вакансії, структурного підрозділу, організації, замовника та події додатковими полями, які використовує Ваша компанія. Для цього використовується розділ Адміністрування->Сервіс->Додаткові поля.

Додаткові поля можливо додати до картки обраного типу об’єкту на вкладку Загальні дані (за замовчуванням картки всіх об’єктів відкриваються на цій вкладці) або можуть бути внесеними на окремо створену вкладку.

У користувача, який буде створювати додаткові поля, повинні бути права на редагування загальних налаштувань.

 

  • Для розширення стандартних карток додатковими полями відкрийте відповідний тип об’єкту.

  • Натисніть на кнопку Новий елемент даних. З’явиться форма з налаштуваннями додаткового поля.

  • Введіть назву та тип нового елемента.

     

Поле Ідентифікатор заповнюється автоматично, його значення змінювати не потрібно. Поле Тип даних визначає, який тип інформації буде зберігатися в даному полі.

Рядок – якщо інформація, яку необхідно буде вносити в дане поле – текст.

Ціле число – якщо інформація є цілим числом.

Дійсне число – якщо інформаціє є числом, та є ймовірність, що також може бути від’ємним числом.

Перемикач – поле в якому проставляється відмітка (так / ні – залежно від того, чи проставлено перемикач).

Дата – якщо до поля буде заноситись інформація у вигляді дати.

Стосовно типу даних Рядок:

  • Перемикач Багаторядкове поле дозволяє створювати поле висотою більше чим один рядок (параметр Висота).
  • В полі Посилання на довідник  можливо вказати прив’язку до будь-якого довідника програми (Спеціальності, Регіони та ін., а також до створеного власного довідника – дивись Власні довідники). Перемикач Вибір з декількох значень надасть користувачу можливість вказувати відразу декілька значень обраного довідника.

Стосовно типу даних Ціле число:

  • В полі Посилання на об’єкт можливо вказати прив’язку до списку об’єктів обраного типу (Структурний підрозділ, Вакансія та ін.). Перемикач Вибір з декількох значень дозволить користувачу вказувати відразу декілька значень із списку об’єктів обраного типу.

Ширина додаткового поля регулюється числовим значенням, яке задається у відсотках, або символами в полі Ширина.

Можливо управляти розташуванням додаткових полів (розташовувати в рядок, додавати пояснюючі надписи, виносити до окремої вкладки).

Для розміщення додаткових полів в один рядок:

  • Створіть управляючий елемент LINE (який вказує початок рядка), натиснувши кнопку Новий управляючий елемент та вказавши тип управляючого елемента Початок рядка.
  • Створіть управляючий елемент LINE-END (який вказує на кінець рядка), натиснувши кнопку Новий управляючий елемент та вказавши тип управляючого елемента Кінець рядка.
  • За допомогою команд Перемістити вище/Перемістити нижче контекстного меню, яке викликається правою кнопкою миші, розташуйте управляючі елементи LINE та LINE-END  таким чином, щоб потрібні поля знаходились між ними.

Для візуально-сенсового структурування додаткових полів можливо додати відповідний надпис над ними:

  • Створіть управляючий елемент LABEL (який вказує на надпис), натиснувши кнопку Новий управляючий елемент та вказавши тип управляючого елемент Надпис.
  • За допомогою команд Перемістити вище/Перемістити нижче контекстного меню, яке викликається правою кнопкою миші, розмістіть управляючий елемент LABEL над тими додатковими полям, над якими повинен знаходити пояснюючий надпис.

Для розміщення в структурі картки додаткових полів на окремій вкладці:

  • Створіть управляючий елемент PAGE (який вказує початок розділу), натиснувши кнопку Новий управляючий елемент та вказавши тип управляючого елемента Початок розділу.
  • Створіть управляючий елемент PAGE-END ((який вказує на кінець розділу), натиснувши кнопку Новий управляючий елемент та вказавши тип управляючого елемента Кінець розділу.
  • За допомогою команд Перемістити вище/Перемістити нижче контекстного меню, яке викликається правою кнопкою миші, розташуйте управляючі елементи PAGE та PAGE-END таким чином, щоб потрібні поля знаходились між ними.

Додані поля будуть доступними у відповідних картках після перезапуску сервера E-Staff (в мережевій версії) та програми.
