Шаблони листів

Програма E-Staff дозволяє відправляти листа кандидатам та замовникам за стандартними шаблонами. Для відправлення листів за шаблонами Ви маєте можливість використовувати кнопки Запрошення та Резюме у замовника в картці відібраного кандидата. Разом із програмою поставляються наступні стандартні шаблони:

·  Запрошення кандидата на інтерв’ю

·  Повідомлення кандидату про призначене інтерв’ю

·  Повідомлення кандидату про призначене інтерв’ю з керівником

·  Повідомлення кандидату про призначене інтерв’ю із зовнішнім замовником

·  Повідомлення кандидату про включення до бази даних

·  Повідомлення кандидату про відмову (після інтерв’ю)

·  Повідомлення кандидату про відмову (призупинення набору на вакансію)

·  Резюме кандидата для замовника

·  Повідомлення кандидату про початок процедури затвердження кандидата

·  Повідомлення кандидату про початок процедури оцінки кандидата

За необхідністю Ви маєте можливість редагувати існуючі шаблони та додавати власні. Для цього використовується розділ Адміністрування->Довідники->Шаблони листів.

  • Для додавання нового шаблону натисніть на кнопку Новий елемент.
  • У новому вікні введіть Назву шаблону (ця назва буде відображатися у списку під час вибору шаблона для відправки), вкажіть Тип отримувача (шаблон для контактної особи, кандидата, користувача, фіксованого адресата або для запису у зовнішньому календарі), тему та текст листа. Якщо було обрано тип отримувача – Користувач, то необхідно перевірити чи у налаштуваннях користувачів (розділ Адміністрування->Доступ->Користувачі) вказаний відповідний співробітник та в картці цих співробітників прописаний E-Mail.

В тілі листа, за аналогією із стандартними шаблонами, Ви маєте можливість використовувати стандартні обчислювальні елементи (ПІБ, вакансія, підпис), на місце яких програма буде ставити фактичні значення.

Примітка: для коректного формування повідомлень з обчислювальними елементами необхідно, щоб на момент створення повідомлень в картці були наявні дані, які підставляє той чи інший обчислювальний елемент.

  • Для додавання файлу, наявність якого обов’язкова при створенні повідомлення за даним шаблоном, напроти надпису Статистичні вкладені файли натисніть кнопку «+» (Додати), вкажіть файл та натисніть кнопку Відкрити.

  • Для додавання файлу з додатку картки, напроти надпису Вкладені файли з картки натисніть кнопку «+» (Додати), вкажіть ім’я файла (можливо використовувати обчислювальні елементи), тип додатку та вихідний об’єкт, з якого додаток вказаного типу буде завантажуватися. Якщо цей додаток може бути не у всіх картках – встановіть перемикач Не додавати файл у випадку його відсутності в картці.

    Якщо необхідно, щоб факт відправлення повідомлення за даним шаблоном фіксувався, встановіть перемикач Реєструвати факт відправлення листа у вигляді події та оберіть необхідну подію.

     

    Перемикач Нова група в меню дозволяє під час відкриття списку шаблонів з картки відокремити даний шаблон разом з тими, які нижче нього в загальному списку шаблонів, від решти.

Вказавши групу в полі Використовувати тільки в середині групи користувачів, можливо обмежити доступ поточного шаблона для користувачів межами вказаної групи.

  • Оскільки у мережевій версії програми всі користувачі використовують загальні шаблони, кожному користувачу необхідно заповнити шаблон свого підпису в розділі Сервіс->Персональні налаштування (пункт Підпис).
