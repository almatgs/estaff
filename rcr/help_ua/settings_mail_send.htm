Відправка пошти

Розділ Відправка пошти служить для налаштування відправлення листів кандидатам та замовникам за шаблонами. Відправлення листів можливе або через вашу поштову програму або напряму через SMTP-сервер. В обох випадках необхідно зробити деякі налаштування E-Staff.

Пункт Метод підготовки одиничних листів – використовується для вибору способу взаємодії з поштовою програмою або вибору відправлення через SMTP-сервер.

Як правило, для більшості поштових програм буде працювати метод Simple MAPI, обраний за замовчуванням. В цьому випадку для нормального функціонування Ваша поштова програма повинна бути вибраною в операційній системи Windows як та, що використовується за замовчуванням. Для цього

·  Запустіть програму Internet Explorer та оберіть команду Властивості браузера з меню Сервіс.

·  В новому вікні оберіть закладку Програми.

·  Натисніть на кнопку Задати програми. У новому вікні – Налаштування доступу програм та замовчуванням.

·  Оберіть Вашу поштову програму, встановіть перемикач Зробити доступною цю програму, натисніть на кнопку ОК.

Цей метод нормально функціонує при використанні поштових програм Outlook Express, Microsoft Outlook, The Bat!.

Якщо при використанні поштової програми Microsoft Outlook формування листа для відправлення не відбувається – оберіть метод підготовки Microsoft Outlook automation.

при використанні поштової програми Lotus Notes можливо використовувати як метод Simple MAPI, так і метод Lotus Notes.

При цьому необхідно, щоб Lotus Notes було встановлено з дистрибутиву та після його інсталяції не була встановлена інша поштова програма. Якщо відправлення листів не працює – знову проінсталюйте Lotus Notes.

Для перевірка взаємодії E-Staff з Вашою поштовою програмою обраним методом натисніть на кнопку Підготувати тестове повідомлення. У вашій поштовій програмі повинно відкритися повідомлення з текстом Test.

Метод підготовки через SMTP може використовуватися у разі відсутності на робочому місці поштової програми. В даному випадку кожен лист створюється та попередньо відкривається в програмі E-Staff. Використання функції відправлення через SMTP можливе за допомогою загальних налаштувань SMTP-серверу компанії (посилання Загальні налаштування SMTP для швидкого переходу до налаштувань) або персональних (наприклад, з використанням налаштувань вихідної пошти та реєстраційні дані свого безкоштовного електронного поштового ящика). В полях Адреса відправника та Ім’я відправника введіть електронну адресу (з якої будуть відправлятися листи) та ім’я відповідно (при використанні загальних налаштувань SMTP листа в цьому випадку будуть відправлятися з електронної адреси користувача, а не з загальної корпоративної адреси, вказаної в налаштуванням автоматичної відправки пошти). Для перевірки коректності введених даних в поле Тестовий E-mail вкажіть свою адресу та натисніть на кнопку Відправити тестове повідомлення. При коректно введених даних Ви отримаєте повідомлення з текстом Test.

Для правильного налаштування можливо знадобиться допомога Вашого системного адміністратора. В окремих випадках налаштування може зробити тільки системний адміністратор.
