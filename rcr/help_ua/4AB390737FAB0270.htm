Підстави для появи вакансії

Довідник Підстави для появи вакансії містить перелік підстав появи вакансії, які використовуються в системі, для швидкого додавання їх до поля Підстави появи в картці вакансії.

За необхідністю Ви маєте можливість редагувати та видаляти існуючі елемента даного довідники та додавати власні.
