Картка організації

Картка організації в програмі E-Staff Рекрутер має наступну структуру:

  • В лівій частині картки (під змістом розділу) розміщений перелік ролей, які можуть бути надані організації в залежності від функцій, які виконує дана організація по відношенню до Вашої компанії. Також там міститься прапор Джерело надходження кандидатів, встановлення якого автоматично створює картку нового джерела у довіднику Кандидати->Джерела кандидатів.

  • Розділ Загальна інформація містить загальну інформацію про організацію.

  • Поле Номер заповнюється програмою автоматично. Але Ви можете виправити його власноруч. Якщо Вас не влаштовує спосіб нумерації, який пропонує програма, Ви маєте змогу змінити його в розділі Адміністрування->Об’єкти.

  • Поле Умовне найменування містить найменування організації, яке буде використовуватися для внутрішнього використання в програмі, наприклад Главстройпроект.

  • Поле Повне (офіційне) найменування містить найменування організації для використання у офіційних документах. Наприклад, ОАО «Главстройпроект».

  • Поле Альт. найменування містить альтернативне найменування організації, за яким дану організацію можливо буде знайти. Як правило, це скорочене або іноземне найменування. Наприклад, ГСП.

  •  Поле Адреса містить фактичну адресу організації.

  • Поле Регіон – регіон, який обирається з довідника регіонів, в якому розміщена організація.

  • Поля Телефон, Факс, E-Mail, Web-сайт – містять загальну контактну інформацію організації. Дана інформація не прив’язана до визначеної особи.

  • Поле Контактні особи – містить перелік контактних осіб даної організації.

  • Поля Сфера діяльності, Розмір, Категорія класифікують організацію за відповідними довідниками, які налаштовуються в програмі.

  • Поле Дата останнього контакту містить дату останнього контакту із організацією. Якщо дане поле у будь-якої контактної особи становиться більше, ніж таке саме поле за організацією, поле Дата останнього контакту по організації оновлюється до відповідного значення.

  • Поля Всього вакансій, Активний вакансій відображають кількість вакансій (загальну та в роботі відповідно) по клієнту (тільки для режиму «Рекрутингове агентство»).

  • Поле Дод. інформація може містити будь-яку довільну інформацію.

  • Розділ Реквізити містить реквізити організації як юридичної особи.

  • Розділ Всі вакансії містить перелік відкритих на даний проміжок часу вакансій по клієнту (тільки для режиму «Рекрутингове агентство»).

  • Розділ Події за кандидатами містить перелік всіх подій за кандидатами, яких розглядають на вакансії за даним клієнтом (тільки для режиму «Рекрутингове агентство»).

  • Розділ Додатки може містити будь-які додатки до картки організації.
