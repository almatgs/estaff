Вимоги до апаратно-програмного забезпечення

·  Робоче місце

  • Операційна система Microsoft Windows XP, Server 2003, Vista, Server 2008, Windows 7, Windows 8
  • CPU від 1.2 GHz
  • 1 Gb оперативної пам’яті.
  • Microsoft Office 2003 чи старша версія (для імпорту резюме з документів Microsoft Word або інтеграції з календарем Microsoft Outlook), або OpenOffice.org 2.3 або старша версія (для імпорту резюме з документів OpenOffice.org).

·  Сервер (для мережевої версії, до 10 робочих місць)

  • Операційна система Microsoft Windows Server 2003, Server 2008, Server 2012 
  • CPU від 2.2 GHz
  • 4 Gb оперативної пам’яті.

·  Сервер (для мережевої версії, 10 та більше робочих місць)

  • Операційна система Microsoft Windows Server 2003, Server 2008, Server 2012 
  • CPU від 2.2 GHz x 2
  • 8 Gb оперативної пам’яті.
