Загальні принципи роботи з розділами програми

Всі розділи програми, які містяться в лівій частині головного вікна, побудовані за єдиним принципом.

В правій частині екрана відображається перелік об’єктів, які відповідають обраному розділу (підрозділи, вакансії, кандидати та інше).

  • Для зміни набору динамічних фільтрів для поточного списку об’єктів натисніть Вид-Фільтри.
  • Для зміни набору стовпчиків у поточному списку об’єктів натисніть Вид-Стовпці.

За допомогою команд Перемістити вище/Перемістити нижче контекстного меню, яке викликається правою кнопкою миші, Ви маєте можливість змінити розташування фільтрів/стовпчиків.

За замовчуванням зміна набору фільтрів та стовпчиків відбувається тільки для даного робочого місця (режим Налаштувати тільки для мене). Для централізованого внесення змін встановіть режим Налаштувати для всіх користувачів.

  • Для зміни порядку сортування об’єктів натисніть курсором миші на назві колонки, за якою повинне відбуватись сортування.
  • Для повернення до попереднього порядку сортування – знову натисніть курсором миші на назві колонки.
  • Для того, щоб відкрити картку об’єкта, двічі натисніть курсором миші на відповідному рядку у списку.
  • Для видалення картки об’єкта – виділіть відповідний рядок в списку, потім натисніть кнопку Delete.

Також для відкриття чи видалення об’єкту можливо використовувати контекстне меню, яке виникає після натискання правою кнопкою миші на рядку у списку.

Над списком відображається набір полів, за якими можливо здійснювати фільтрацію об’єктів у списку. Першим завжди буде поле Пошук по тексту, яке дозволяє ввести будь-яке ключове слово (чи його фрагмент), яке повинно міститися у картці. Більш детальніше викладено у розділі Пошук за ключовими словами.  

  • Для фільтрування записів введіть потрібний набір значень фільтрів у верхній частині екрана над списком. Для фільтрів, які посилаються на довідник (наприклад, Регіон, Спеціальність), можливо вибрати відразу декілька значень, вказуючи їх по черзі, утримуючи при цьому клавішу Ctrl.

При цьому, у випадку вибору фільтру з випадаючим списком значень, фільтр застосовується відразу після вибору значення.

У випадку фільтра із введенням тексту – для його застосування необхідно натиснути кнопку Знайти, або, знаходячись у текстовому полі, натиснути на клавішу Enter.

  • Для анулювання окремного фільтру з випадаючим списком значень необхідно правою кнопкою миші натиснути на цьому фільтрі та натиснути Очистити поле. Для анулювання усіх фільтрів, які використовуються, необхідно натиснути на вільній області правою кнопкою миші та обрати Очистити форму.
