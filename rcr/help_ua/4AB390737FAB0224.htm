Довідники

Розділ Адміністрування->Довідники містить довідники, які використовуються програмою. Загальні для всіх об’єктів довідники розміщено у підрозділі Загальні, всі інші – які використовуються тільки в певних об’єктах (контрагенти, вакансії, кандидати) – в окремих відповідних підрозділах.

Довідники, в залежності від призначення, можуть бути однорівневими (наприклад, Бренди, Статті витрат), чи ієрархічними.

Вибір будь-якого елемента з довідника під час роботи з програмою відбувається з випадаючого списку або з діалогового вікна з функцією пошуку. Спосіб вибору елемента кожного з довідників задається в розділі Адміністрування->Довідники у вікні його властивостей, яке викликається правою кнопкою миші (Властивості довідника) у вільній області біля поля Пошук за текстом. Також там же можливо увімкнути режим сортування елементів довідника за їх назвою.

  • Для додавання нового елемента довідника натисніть на кнопку Новий елемент. Для ієрархічних довідників в полі Батьківський елемент вкажіть елемент рівня, який знаходиться вище.

Стандартні елементи довідників, які були змінені користувачем, помічаються у розділі Тип елемента символом (*). Для повернення елемента до попереднього вигляду, яке прописане у дистрибутиві програми, натисніть на даному елементі правою кнопкою миші, оберіть пункт Повернути до попереднього вигляду та дотримуйтесь вказівок програми.

  • Для додавання нового елемента довідника на основі існуючого, в картці наявного елемента натисніть на кнопку Файл->Створити дублікат картки.
  • Для видалення елемента довідника оберіть його з переліку та, натиснувши правою кнопкою миші, оберіть пункт Видалити, або на клавіатурі натисніть клавішу Delete.
